# Unsupervised Learning of Visual Features by Contrasting Cluster Assignments (Caron et al., 2020)

**arxiv**: 2006.09882 | **关键词**: SwAV, 在线聚类, 交换预测, multi-crop | Inria / Facebook AI Research | NeurIPS 2020

---

## 一、核心问题

对比学习方法需要大量成对特征比较（负样本），计算代价高且需大 batch 或 memory bank。能否通过聚类分配而非直接特征比较来实现对比学习，同时保持在线性和可扩展性？此外，如何在不增加计算和内存开销的前提下利用更多图像视图来提升性能？

## 二、方法

### 2.1 SwAV：交换分配预测

SwAV 的核心思想是在线聚类并强制不同增强视图的聚类分配保持一致，而非直接比较特征。

给定图像的两个增强视图的特征 $z_t$ 和 $z_s$，将它们映射到 $K$ 个可学习原型向量 $\{c_1, \ldots, c_K\}$（矩阵 $C$），得到编码 $q_t$ 和 $q_s$。交换预测损失：

$$\mathcal{L}(z_t, z_s) = \ell(z_t, q_s) + \ell(z_s, q_t)$$

其中 $\ell(z, q)$ 是特征 $z$ 与编码 $q$ 之间的交叉熵：

$$\ell(z_t, q_s) = -\sum_k q_s^{(k)} \log p_t^{(k)}, \quad p_t^{(k)} = \frac{\exp(z_t^\top c_k / \tau)}{\sum_{k'} \exp(z_t^\top c_{k'} / \tau)}$$

### 2.2 在线编码计算

编码 $Q = [q_1, \ldots, q_B]$ 通过优化传输问题获得，确保每个原型被均匀使用（等分约束）：

$$\max_{Q \in \mathcal{Q}} \text{Tr}(Q^\top C^\top Z) + \varepsilon H(Q)$$

约束集：

$$\mathcal{Q} = \left\{ Q \in \mathbb{R}_+^{K \times B} \mid Q \mathbf{1} = \frac{1}{K}\mathbf{1}, \quad Q^\top \mathbf{1} = \frac{1}{B}\mathbf{1} \right\}$$

等分约束防止所有样本被分配到同一原型（避免平凡解）。

### 2.3 Sinkhorn-Knopp 算法

连续解通过 Sinkhorn-Knopp 算法高效计算：

$$Q^* = \text{Diag}(u) \exp\left(\frac{C^\top Z}{\varepsilon}\right) \text{Diag}(v)$$

仅需 3 次迭代即可收敛，GPU 上 4K 特征对齐 3K 编码仅需 35ms。

### 2.4 Multi-crop 策略

使用 2 个标准分辨率裁剪 + $V$ 个低分辨率小裁剪，损失推广为：

$$\mathcal{L}(z_{t_1}, z_{t_2}, \ldots, z_{t_{V+2}}) = \sum_{i \in \{1,2\}} \sum_{v=1}^{V+2} \frac{1}{v \neq i} \ell(z_{t_v}, q_{t_i})$$

- 编码仅从全分辨率裁剪计算（小裁剪信息不完整，影响编码质量）
- 小裁剪仅用于特征提取端，提供局部-全局对应关系
- 不增加显著计算和内存开销

### 2.5 小 batch 处理

当 batch 太小无法等分到 $K$ 个原型时，使用之前 batch 的特征（约 3K 个）扩展 $Z$，仅需存储约 15 个 batch（远少于 MoCo 的 250 个 batch / 65K 特征）。

## 三、实验结果

**ImageNet 线性评估（ResNet-50）**：

| 方法 | Top-1 |
|------|-------|
| SimCLR | 69.3% |
| MoCo v2 | 71.1% |
| **SwAV** | **75.3%** |
| 监督 | 76.5% |

SwAV 比 MoCo v2 高 4.2%，仅低监督 1.2%。

**Multi-crop 的通用提升**：
- SimCLR + multi-crop：+2.4%
- DeepCluster-v2 + multi-crop：+4.1%
- SwAV + multi-crop：+4.0%

**半监督学习**：1% 标签 53.9%，10% 标签 70.2%，优于 SimCLR。

**迁移学习**：在 Places205、VOC07、iNat18 线性分类及 VOC/COCO 目标检测上均超过监督预训练——首个在所有迁移任务上超越监督预训练的自监督方法。

**小 batch（256）**：SwAV 达 74.3%（2x224+6x96, 400 epochs），仅存储 3,840 特征 vs MoCo v2 的 65,536。

**未整理数据预训练**：在 1B 随机 Instagram 图像上预训练，SwAV 在 ImageNet 上仍比 SimCLR 高约 6%。

## 四、个人评价

### 影响

SwAV 是自监督视觉表示学习的重要里程碑，通过"对比聚类分配"的创新范式摆脱了对大量负样本对的依赖，同时保持在线性和可扩展性。Multi-crop 策略是一个简洁但极其有效的通用增强方案。SwAV 是首个在所有迁移任务上全面超越监督预训练的自监督方法，对领域产生了深远影响。

### 局限

1. 原型数量 $K$ 的选择对性能有一定影响，虽论文显示 3K-100K 范围内差异不大（<0.3%），但需调参
2. Sinkhorn 算法的等分约束假设数据均匀分布于各原型，对长尾分布数据可能不合适
3. 编码计算（虽快）仍引入额外计算，比 SimCLR 的纯对比损失稍慢
4. 多节点分布式训练时，Sinkhorn 的行/列归一化统计量需跨 GPU 同步
