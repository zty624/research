# Hierarchical VAEs Know What They Don't Know (Havtorn et al., 2021)

**arxiv**: 2102.08248 | **层次VAE, OOD检测, 似然比** | DTU / Corti AI | ICML 2021

## 一、核心问题

深度生成模型虽然是 SOTA 密度估计器，却经常给**分布外（OOD）数据赋予更高的似然**。这一看似矛盾的现象引发了对密度估计质量的严重质疑。

核心问题：
1. 为什么生成模型会给 OOD 数据更高的似然？
2. 能否利用层次 VAE 的结构特性设计更好的 OOD 检测方法？

## 二、方法

### 1. OOD 失败的根因分析

作者通过实验揭示：

- **低层特征跨数据集高度共享**：分别训练在 FashionMNIST 和 MNIST 上的 HVAE，底层特征之间的相关性极高，高层特征才出现分化
- **OOD 数据的重建质量出奇地好**：FashionMNIST 模型重建 MNIST 数据几乎和重建训练数据一样好
- **高层语义特征才具有区分性**：将低层推理分布替换为条件先验后，OOD 重建质量急剧下降

**核心假设**：低层潜变量学到了通用的低级特征（如边缘），这些特征在 OOD 数据上也成立，导致高似然。OOD 数据对低层特征是 in-distribution 的，但对语义特征是 out-of-distribution 的。

### 2. $L_{>k}$ 界

对 ELBO 的变体，将最低 $k$ 层潜变量从条件先验 $p_\theta(z_{\leq k}|z_{>k})$ 采样，而非近似后验 $q_\phi(z|x)$：

$$L_{>k} = \mathbb{E}_{p_\theta(z_{\leq k}|z_{>k}) q_\phi(z_{>k}|x)} \left[\log \frac{p_\theta(x|z) p_\theta(z_{>k})}{q_\phi(z_{>k}|x)}\right]$$

当 $k=0$ 时退化为标准 ELBO。增大 $k$ 使界变松，但更依赖语义特征。

### 3. 似然比分数 $\text{LLR}_{>k}$

$$\text{LLR}_{>k}(x) = L(x) - L_{>k}(x)$$

将 ELBO 和 $L_{>k}$ 用精确形式写出：

$$L = \log p_\theta(x) - D_{KL}(q_\phi(z|x) \| p_\theta(z|x))$$

$$L_{>k} = \log p_\theta(x) - D_{KL}(p_\theta(z_{\leq k}|z_{>k}) q_\phi(z_{>k}|x) \| p_\theta(z|x))$$

相减后 $\log p_\theta(x)$ 消去，**LLR 纯粹在潜空间中度量**：

$$\text{LLR}_{>k}(x) = -D_{KL}(q_\phi(z|x) \| p_\theta(z|x)) + D_{KL}(p_\theta(z_{\leq k}|z_{>k}) q_\phi(z_{>k}|x) \| p_\theta(z|x))$$

这消除了数据空间似然对 OOD 检测的干扰。

### 4. 重要性加权增强

使用 IWAE 的 $S$ 个重要性采样，当 $S \to \infty$ 时：

$$\text{LLR}^S_{>k}(x) \to D_{KL}(p(z_{\leq k}|z_{>k}) q(z_{>k}|x) \| p(z|x))$$

### 5. 数学分析：低层特征主导似然

对于确定解码器，边际似然可写为：

$$\log p(x) = \log p(z) - \frac{1}{2} \sum_{i=1}^{L} \log \det J_i^T J_i$$

由于瓶颈结构 $d_0 \geq d_1 \geq \ldots \geq d_L$，低层 Jacobian 行列式对似然的贡献远大于高层。Gaussian Jacobian 分析表明 $\det J_i^T J_i$ 随维度超指数增长，证实**低层特征主导了似然值**。

## 三、实验结果

### 主要 OOD 检测结果

| 方法 | FMNIST→MNIST AUROC↑ | CIFAR10→SVHN AUROC↑ |
|------|---------------------|---------------------|
| ELBO ($L_{>0}$) | 0.455 | 0.083 |
| $L_{>1}$ + HVAE | 0.877 | 0.097 |
| $\text{LLR}_{>1}$ + HVAE | **0.986** | - |
| $\text{LLR}_{>2}$ + HVAE | - | **0.811** |
| $\text{LLR}_{>2}$ + BIVA | - | **0.964** |
| Likelihood Regret | 0.942 | 0.855 |
| WAIC (5 Glow) | 0.752 | 0.432 |

- 在 FashionMNIST/MNIST 上，$\text{LLR}_{>1}$ + 250 重要性采样达到 AUROC 0.998
- 在 CIFAR10/SVHN 上，$\text{LLR}_{>2}$ + BIVA 达到 AUROC 0.964
- **完全无监督**，无需 OOD 数据或标签
- 计算复杂度 $O(DL)$，支持批处理

## 四、个人评价

**优点**：
1. 对 OOD 失败现象的解释令人信服——低层特征跨域共享是根本原因，这比单纯说"似然不可靠"更深刻
2. LLR 分数的推导优雅——$\log p_\theta(x)$ 项相消后纯粹在潜空间操作，规避了数据空间似然的问题
3. 数学分析（Jacobian 行列式维度依赖性）为低层特征主导似然提供了理论支撑
4. 完全无监督，仅需单个模型，无需 OOD 样本或标签

**不足**：
1. 方法依赖于层次 VAE 的特定结构，不能直接应用于自回归模型或 flow-based 模型
2. $k$ 值的选择需要参考 in-distribution 数据的经验分布
3. 当 OOD 差异在低层统计特征而非高层语义时，方法可能失效（作者承认了这一点）
4. BIVA 模型训练耗时约一周（单 GPU）

**影响**：这篇工作深化了对深度生成模型 OOD 失败的理解，所提的 LLR 分数启发了后续大量关于层次特征与 OOD 检测的研究。
