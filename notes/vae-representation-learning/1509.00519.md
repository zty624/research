# Importance Weighted Autoencoders (Burda, Grosse & Salakhutdinov, 2015)

**arxiv**: 1509.00519 | **IWAE**

---

## 一、理论推导

### 1. VAE 回顾

标准 VAE 目标 (ELBO)：

$$\log p(x) \geq \mathcal{L}(x) = \mathbb{E}_{q(h|x)}\left[\log \frac{p(x,h)}{q(h|x)}\right]$$

其中 $h = \{h_1, \ldots, h_L\}$ 为多层隐变量。多层级联结构：
- 生成：$p(x|\theta) = \sum_h p(h_L|\theta) p(h_{L-1}|h_L,\theta) \cdots p(x|h_1,\theta)$
- 推断：$q(h|x) = q(h_1|x) q(h_2|h_1) \cdots q(h_L|h_{L-1})$

### 2. IWAE 目标函数

核心思想：用 $k$ 个独立采样的重要性权重构造更紧的下界：

$$\mathcal{L}_k(x) = \mathbb{E}_{h_1,\ldots,h_k \sim q(h|x)}\left[\log \frac{1}{k}\sum_{i=1}^{k} \frac{p(x, h_i)}{q(h_i|x)}\right] = \mathbb{E}_{h_1,\ldots,h_k \sim q(h|x)}\left[\log \frac{1}{k}\sum_{i=1}^{k} w_i\right]$$

其中 $w_i = p(x,h_i)/q(h_i|x)$ 为未归一化重要性权重。

**这是 ELBO 的下界**，由 Jensen 不等式直接可得：

$$\mathcal{L}_k = \mathbb{E}\left[\log \frac{1}{k}\sum_{i=1}^k w_i\right] \leq \log \mathbb{E}\left[\frac{1}{k}\sum_{i=1}^k w_i\right] = \log p(x)$$

### 3. 关键定理：单调收紧

**定理 1**：对于所有 $k$，

$$\log p(x) \geq \mathcal{L}_{k+1} \geq \mathcal{L}_k$$

且当 $p(h,x)/q(h|x)$ 有界时，$\lim_{k \to \infty} \mathcal{L}_k = \log p(x)$。

**证明思路**：
- 下界性：Jensen 不等式
- 单调性：从 $k$ 个样本中均匀抽取 $m$ 个子集，利用 $\mathbb{E}_I[\sum_{j \in I} a_j / m] = \sum_j a_j / k$ 和 Jensen 不等式
- 收敛性：大数定律，$M_k = \frac{1}{k}\sum_{i=1}^k w_i \to \mathbb{E}[w_i] = p(x)$ a.s.

### 4. 梯度估计

利用重参数化技巧，$h_i = h(\epsilon_i, x, \theta)$，$\epsilon_i \sim \mathcal{N}(0, I)$：

$$\nabla_\theta \mathcal{L}_k(x) = \mathbb{E}_{\epsilon_1,\ldots,\epsilon_k}\left[\sum_{i=1}^{k} \tilde{w}_i \nabla_\theta \log w(x, h(\epsilon_i, x, \theta), \theta)\right]$$

其中 $\tilde{w}_i = w_i / \sum_{j=1}^k w_j$ 为归一化重要性权重。

**关键洞察**：当 $k=1$ 时，$\tilde{w}_1 = 1$，退化为标准 VAE 更新。$k > 1$ 时，高质量样本（$w_i$ 大）获得更大梯度权重，低质量样本被自动抑制。

### 5. 计算技巧

- 标准实现：$k$ 个样本并行处理（复制 mini-batch $k$ 倍），前向+反向传播各 $k$ 次
- **降低成本**：只做 $k$ 次前向传播计算权重，按 $\tilde{w}_i$ 概率选一个样本做反向传播。计算量减少约 3 倍（代价是方差增大）

### 6. 活跃维度分析

定义隐维度 $u$ 的活跃度：$A_u = \text{Cov}_x[\mathbb{E}_{u \sim q(u|x)}[u]]$，若 $A_u > 10^{-2}$ 则为活跃。

实验发现 VAE 只使用很少的活跃维度，IWAE 使用更多。关键实验：将 VAE 模型切换到 IWAE 目标继续训练，活跃维度和 log-likelihood 都上升；反向操作则下降。说明维度退化是**目标函数驱动**而非优化问题。

## 三、重要 Idea

1. **重要性加权构造更紧下界**：核心贡献是 $\mathcal{L}_k$ 目标。通过取 $k$ 个样本的权重平均再取对数（而非先取对数再平均），利用 Jensen 不等式的方向差异，得到比 ELBO 更紧的边界。$k$ 越大越紧，极限收敛到真实 log-likelihood。

2. **归一化权重作为梯度加权**：梯度中 $\tilde{w}_i$ 使得对"好"样本的梯度更大，对"差"样本的梯度被抑制。这比 VAE 的等权平均更合理 —— VAE 即使大部分概率质量在好的解释上，也会被少数差样本拖累。

3. **解决 VAE 的过度正则化问题**：VAE 的 ELBO 严厉惩罚无法解释数据的后验样本，导致模型倾向学习简单表示使得近似后验足够准确。IWAE 允许近似后验更"松散"，只要部分样本能解释数据即可，释放了模型的表达能力。

4. **活跃维度诊断**：提出了一种量化隐空间利用率的方法，发现 VAE 只激活很少维度。这不仅是 IWAE 论文的副产品，更是理解 VAE 行为的重要工具。

5. **VAE 的特例关系**：$k=1$ 时 IWAE = VAE，两者完全等价。这使得 IWAE 成为 VAE 的自然推广，无需修改架构，只需改目标函数。

## 评价

IWAE 是对 VAE 的优雅改进。理论上，$\mathcal{L}_k$ 的单调收紧性质非常漂亮；实践上，它确实让模型学到更丰富的表示。但 $k$ 增大带来线性增加的计算成本（即使有采样技巧也只减少 3 倍），且论文也指出 $k > 50$ 后收益递减。后来的工作（如 VIME、AmpVI）在此基础上进一步改进了重要性权重的利用方式。活跃维度的分析视角对理解 VAE 行为有长期价值。
