## 一、理论推导

### 1. 模型架构

VQ-VAE 与标准 VAE 的两个关键区别：
- **离散隐变量**：编码器输出离散 code 而非连续向量
- **学习先验**：先验不是静态的（如标准高斯），而是训练后学习的

定义嵌入空间（codebook）$e \in \mathbb{R}^{K \times D}$，其中 $K$ 为离散空间大小，$D$ 为每个嵌入向量维度。

### 2. 离散化过程

编码器输出 $z_e(x)$，通过最近邻查找映射到 codebook：

$$q(z = k | x) = \begin{cases} 1 & \text{for } k = \arg\min_j \|z_e(x) - e_j\|_2^2 \\ 0 & \text{otherwise} \end{cases}$$

解码器输入为对应的嵌入向量：

$$z_q(x) = e_k, \quad \text{where } k = \arg\min_j \|z_e(x) - e_j\|_2^2$$

### 3. 训练目标

总损失函数包含三项：

$$L = \underbrace{\log p(x|z_q(x))}_{\text{重建损失}} + \underbrace{\|\text{sg}[z_e(x)] - e\|_2^2}_{\text{VQ 损失}} + \underbrace{\beta \|z_e(x) - \text{sg}[e]\|_2^2}_{\text{承诺损失}}$$

其中 $\text{sg}[\cdot]$ 为 stopgradient 算子（前向恒等，反向梯度为零）。

- **重建损失**：优化编码器和解码器。由于最近邻查找不可微，使用 straight-through estimator：将解码器输入 $z_q(x)$ 的梯度直接传给编码器输出 $z_e(x)$
- **VQ 损失**：将 codebook 向量 $e_i$ 拉向编码器输出（类似 K-means 更新），仅更新 $e$
- **承诺损失**：约束编码器输出不远离已选的 codebook 向量，仅更新编码器。$\beta = 0.25$

### 4. KL 项

因为假设均匀先验 $p(z) = 1/K$，后验 $q(z|x)$ 是 one-hot 的，KL 散度为常数 $\log K$，不依赖编码器参数，训练时可忽略。

### 5. EMA 更新（替代方案）

可用指数移动平均替代 VQ 损失更新 codebook：

$$N_i^{(t)} := N_i^{(t-1)} \cdot \gamma + n_i(1-\gamma)$$
$$m_i^{(t)} := m_i^{(t-1)} \cdot \gamma + \sum_{j} z_{i,j}^{(t)} (1-\gamma)$$
$$e_i^{(t)} := m_i^{(t)} / N_i^{(t)}$$

其中 $\gamma = 0.99$，$n_i$ 为 mini-batch 中分配到 $e_i$ 的编码器输出数量。

### 6. 两阶段训练

1. 训练 VQ-VAE（编码器 + 解码器 + codebook），先验为均匀分布
2. 冻结 VQ-VAE，在离散隐空间上训练自回归先验（PixelCNN / WaveNet）

### 7. Log-likelihood 估计

$$\log p(x) = \log \sum_k p(x|z_k) p(z_k) \approx \log p(x|z_q(x)) p(z_q(x))$$

因为解码器只在 $z = z_q(x)$ 处分配概率质量，其他 $z$ 的贡献近似为零。

| 项目 | 要求 |
|------|------|
| **数据集** | CIFAR-10 (32×32), ImageNet 128×128, VCTK 语音, DeepMind Lab |

| **硬件** | 单 GPU, ~12-24 小时 (CIFAR-10), 更长 (ImageNet) |

CIFAR-10 最小复现约 200-300 行 PyTorch。

## 三、重要 Idea

1. **向量量化 (VQ) 实现离散隐变量**：通过最近邻查找将连续编码映射到离散 codebook，用 straight-through estimator 绕过不可微的量化操作。这个方法比 Gumbel-Softmax 更简单、方差更低。

2. **避免 posterior collapse**：这是 VQ-VAE 最重要的贡献。标准 VAE 搭配强自回归解码器时，隐变量常被忽略（posterior collapse）。VQ-VAE 的后验是确定性的 one-hot，KL 项为常数，因此不存在 KL 与重建的权衡 —— 隐变量不会被 KL 正则化"压死"。实验验证了两阶段 VQ-VAE 中隐变量仍然有效。

3. **离散表示的天然优势**：语言、语音等天然离散；离散 code 更适合自回归先验建模；压缩率极高（ImageNet 128×128×3 → 32×32×1, 压缩约 42.6 倍）。

4. **两阶段训练分离表示学习和生成**：VQ-VAE 学习离散表示，PixelCNN/WaveNet 学习先验。这种分离简化了训练，但可能不是最优的（论文也承认联合训练是未来方向）。

5. **无监督语音分析**：从原始波形学到接近音素的离散表示（49.3% 映射准确率 vs 随机 7.2%），并能做说话人转换。这是论文最令人惊讶的结果。

## 评价

VQ-VAE 开创了离散表示学习的新范式，直接影响后续 VQGAN、DALL-E 等工作。核心贡献是解决了 VAE 的 posterior collapse 问题，并证明了离散表示在多模态上的有效性。局限包括：(1) 两阶段训练非端到端最优；(2) straight-through estimator 是近似梯度；(3) 重建质量仍偏模糊（后来 VQGAN 用 GAN 判别器解决）。但作为概念验证，这篇论文的影响远超其技术细节。
