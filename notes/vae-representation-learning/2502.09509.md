# EQ-VAE: Equivariance Regularized Latent Space for Improved Generative Image Modeling

- 机构: Archimedes/Athena Research Center, NTU Athens, valeo.ai, University of Crete, IACM-Forth

## 1. 理论推导

### 等变性约束的数学定义

给定输入图像 x: R^2 -> R^c 定义在二维坐标 p = [u, v]^T 上，空间变换 tau in R^{2x2} 作用于坐标：

x_tau(p) = x(tau^{-1} p)

简记为 tau ◦ x。编码器 E 的等变性定义为：

E(tau ◦ x) = tau ◦ E(x),  对所有 x in X 成立

即：对输入施加变换后编码，应等价于对编码后的隐表示施加同一变换。当前 SOTA 自编码器（SD-VAE 等）不满足此性质，导致语义相似的输入（x 与 tau ◦ x）被编码到不满足变换关系的隐码 E(x) 与 E(tau ◦ x)，使得隐流形不必要地复杂。

### 显式正则化及其失败

最直接的等变约束是将式 (3) 作为损失项：

L_explicit(x) = ||tau ◦ E(x) - E(tau ◦ x)||_2^2

但实验表明这会导致模式坍塌（latent collapse）：编码器将所有输入映射为常数 E(x) = c，等变误差虽降至极低，但重建质量和生成性能严重退化（R-FID 从 0.90 恶化至 117.93，G-FID 从 43.5 恶化至 141.3）。即使加入 stop-gradient 也无法解决。

### 隐式正则化（EQ-VAE 核心方法）

为避免模式坍塌，EQ-VAE 采用隐式方法：将变换后的隐表示的重建 D(tau ◦ E(x)) 与变换后的输入 tau ◦ x 对齐。修改后的训练目标为：

L_EQ-VAE(x, tau) = L_rec(tau ◦ x, D(tau ◦ E(x))) + lambda_gan L_gan(D(tau ◦ E(x))) + lambda_reg L_reg

核心思想：不直接约束编码器的等变性，而是通过重建损失和对抗损失的丰富监督信号，间接迫使编码器产生等变的隐表示。当 tau 为恒等变换时，目标退化为原始 VAE 目标。

### 正则化强度控制

以概率 p_alpha 采样恒等变换（即使用原始目标），以概率 1 - p_alpha 使用 EQ-VAE 目标。默认 p_alpha = 0.5。总目标：

L_total(x) = L_VAE(x),         if p < p_alpha
             L_EQ-VAE(x, tau),  if p >= p_alpha

### 变换设计

采用各向异性缩放与旋转的复合变换 tau = S(s_x, s_y) · R(theta)：

S(s_x, s_y) = [[s_x, 0], [0, s_y]],   R(theta) = [[cos theta, -sin theta], [sin theta, cos theta]]

采样范围：0.25 < s_x, s_y < 1（均匀采样），theta in {pi/2, pi, 3pi/2}（90 度倍数的旋转，避免角落伪影）。下采样使用双三次插值。消融实验表明缩放等变性对生成性能的贡献大于旋转等变性，两者组合效果互补最优。各向异性缩放正则化更强但会轻微损害重建质量，因此默认使用旋转 + 各向同性缩放组合。

### 等变性如何改善隐空间结构

等变约束使语义相似但经过空间变换的输入在隐空间中保持简单的变换关系，消除了隐流形中不必要的非线性结构。论文用 Intrinsic Dimension (ID) 量化隐空间复杂度：SD-VAE 的 ID 为 62.2，加上 EQ-VAE 后降至 38.9。ID 的下降与 G-FID 的改善强相关，说明隐分布变得更简单、更容易被生成模型学习。从 PCA 可视化看，EQ-VAE 的隐表示在前三个主成分上更平滑。

### 等变性与解耦表征的关系

等变性与解耦表征（disentanglement）有深刻联系。当隐空间满足等变性时，空间变换因素（缩放、旋转）被"解耦"出来：它们在隐空间中以与输入空间相同的线性方式作用，而非被编码器纠缠到复杂的非线性映射中。这意味着生成模型不需要额外学习变换的非线性效应，大幅降低了学习难度。与之对比，Sinha & Dieng (2021) 提出一致性正则化强制隐码在空间变换下不变（invariance），而 EQ-VAE 选择等变（equivariance）而非不变——保留变换信息但使其在隐空间中保持结构化，这对生成任务更合适。

### 对离散自编码器的扩展

对 VQ-GAN 等离散自编码器，在量化之前的连续特征 E(x) 上施加变换 tau，然后正常进行量化。无需其他修改。

## 2. 最小实验复现资源

- **数据集**:
  - 自编码器微调：OpenImages（遵循 LDM 框架）
  - 生成模型训练：ImageNet 256x256（DiT/SiT/REPA），ImageNet（MaskGIT）
- **自编码器模型**: SD-VAE (4 channels, f=8), SDXL-VAE (4ch, f=8), SD3-VAE (16ch, f=8), SD-VAE-16 (16ch, f=16), VQ-GAN (256 codebook, f=16)
- **生成模型**: DiT-B/2 (130M), DiT-XL/2 (675M), SiT-B/2, SiT-XL/2, REPA (SiT-XL/2), MaskGIT
- **硬件**: NVIDIA A100 GPU（评估用）；训练使用 GRNET HPC 和 GENCI-IDRIS 集群
- **训练时间**:
  - EQ-VAE 微调：5 epochs，batch size 10，在 OpenImages 上（约 10K 步）；即使 1 个 epoch 即可将 G-FID 从 43.5 降至 36.7
  - DiT-XL/2 原始训练需 7M 步达到 G-FID 9.6，配合 EQ-VAE 仅需 1.5M 步达到 8.8（约 7 倍加速）
  - REPA (SiT-XL/2) 原始需 4M 步达到 G-FID 5.9，配合 EQ-VAE 仅需 1M 步（约 4 倍加速）
  - MaskGIT 原始需 300 epochs，配合 EQ-VAE 130 epochs 即可达到同等性能（约 2 倍加速）

## 3. 重要idea

1. **等变性作为 VAE 隐空间的正则化手段**：这是本文最核心的洞察——现有自编码器的隐表示对缩放和旋转等语义保持变换缺乏等变性，导致隐流形引入不必要的复杂度。通过在训练目标中施加等变约束，可以简化隐空间而不损害重建质量，从而改善下游生成模型的学习效率。

2. **隐式正则化避免模式坍塌**：直接在编码器输出上约束等变性（显式正则化）会导致编码器坍塌为常数函数。EQ-VAE 的关键设计是将等变约束转化为重建目标：要求 D(tau ◦ E(x)) ≈ tau ◦ x。这一隐式方法利用了重建损失和对抗损失的联合监督，在降低等变误差的同时避免模式坍塌。

3. **等变表征与解耦表征的联系**：等变性本质上是一种结构化解耦——将空间变换因素从内容表征中分离出来，使变换在隐空间中以与输入空间相同的几何方式作用。这降低了生成模型需要学习的隐分布复杂度（用 Intrinsic Dimension 度量）。与传统的 KL 正则化（通过约束分布形状来平滑隐空间）不同，等变正则化通过约束隐空间的几何结构来简化它，且不牺牲信息容量。

4. **即插即用、极低成本**：EQ-VAE 不需要修改自编码器架构，也不需要从头训练。仅对预训练自编码器微调 5 个 epoch 即可获得显著效果。同时兼容连续（SD-VAE）和离散（VQ-GAN）自编码器，适用于扩散模型和掩码生成模型等多种生成框架。

5. **缩放等变性比旋转等变性更重要**：消融实验表明，各向同性缩放等变性对生成性能的改善最为显著，旋转等变性次之。两者组合产生互补效果，进一步降低隐空间内在维度并提升生成质量。
