# Representation Learning for Tabular Data: Survey (Jiang et al., 2025)

**arxiv**: 2504.16109

---

## 一、理论推导（综述框架）

### 三类方法的泛化能力分类

1. **专用方法 (Specialized)**：同分布训练和评估
   - 特征级：特征交互建模（交叉网络、注意力）
   - 样本级：实例感知表示（对比学习、原型网络）
   - 目标级：多任务/辅助任务

2. **可迁移方法 (Transferable)**：预训练 + 微调
   - 同质迁移：同类型数据集间迁移
   - 异质迁移：不同特征空间间迁移
   - 跨模态迁移：视觉/语言 → 表格

3. **通用方法 (General/Tabular Foundation Models)**：零样本预测
   - 基于 LLM 的表格理解
   - 无需微调直接应用于新数据集

### 表格数据的挑战

- 异质属性（数值 + 类别 + 混合）
- 稀疏/缺失/噪声
- DNN vs 树模型：DNN 在表格数据上不稳定，树模型仍是强基线

### 与 VAE 的关系

VAE 在表格表示学习中的应用：
- VAE 用于异质表格数据的生成建模
- 条件 VAE 处理混合类型特征
- 变分推断用于表格数据的不确定性量化

综述论文，无单一实验。各方法复现差异大。

## 三、重要 Idea

1. **泛化能力三分法**：Specialized → Transferable → General 的分类清晰反映了表格学习的发展趋势：从单一数据集到迁移学习到基础模型。

2. **DNN 在表格数据上的挑战**：相比图像/文本，DNN 在表格数据上尚未完全超越树模型（XGBoost/LightGBM）。原因：表格数据缺乏空间/序列结构，特征交互不规则，样本量往往不够大。

3. **表格基础模型**：受 LLM 启发，零样本表格预测是前沿方向。但表格数据的异质性使得"通用表示"比视觉/语言更困难。

4. **表示学习的核心：特征交互**：表格数据的表示质量取决于特征交互的建模能力。这比视觉的局部性或语言的序列性更复杂。

## 评价

作为表格表示学习的系统综述，本文覆盖面广、分类清晰。从传统方法到基础模型的递进反映了领域发展趋势。对 VAE 研究者的价值：提供了 VAE 在表格数据中的定位，以及特征交互建模的相关技术参考。局限：各子领域讨论深度不均；缺少对 VAE 类方法的专门深入分析。但作为入门和全景参考很有价值。
