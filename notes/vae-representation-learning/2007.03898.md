**arxiv**: 2007.03898 | **关键词**: 层次化 VAE, 深度可分离卷积, 残差参数化, 谱正则化, 批归一化, Swish | NVIDIA | NeurIPS 2020

## 一、核心问题

VAE 相比归一化流和自回归模型在密度估计上落后，主要瓶颈被认为在统计层面（如近似后验不够灵活）。本文提出正交方向的核心问题：**能否通过精心设计 VAE 的神经网络架构来大幅提升生成性能？** 具体挑战包括：(1) 长程相关性的建模；(2) 深层层次化 VAE 的训练不稳定；(3) 扩展到 256×256 自然图像的可行性。

## 二、方法

### 2.1 深度可分离卷积残差单元

**生成模型残差单元**：
- 1×1 卷积扩展通道数 $E$ 倍 → BN-Swish → 深度可分离 5×5 卷积 → BN-Swish → 1×1 卷积映射回 $C$ 通道
- 深度可分离卷积以极低参数量快速扩大感受野
- 引入 Squeeze-and-Excitation (SE) 通道门控

**编码器残差单元**：
- BN-Swish-Conv(3×3)-BN-Swish-Conv(3×3) 的标准残差块

### 2.2 批归一化 + Swish 激活

先前 SOTA VAE (IAF-VAE, BIVA) 因 BN 噪声损害性能而改用 Weight Norm + ELU。本文发现：
- BN 的负面影响主要在**评估阶段**（running statistics 偏移），而非训练阶段
- 解决方案：调整 BN 动量参数 + 谱正则化确保统计偏差不被放大
- BN + Swish 组合显著优于 WN + ELU

### 2.3 残差正态分布参数化

层次化 VAE 中，近似后验 $q(z_l|x, z_{<l})$ 和先验 $p(z_l|z_{<l})$ 均为因子化高斯。传统参数化直接预测 $(\mu, \sigma)$，本文提出**残差参数化**：

$$q(z_l|x, z_{<l}) = \mathcal{N}(\mu_p + \Delta\mu_q, \sigma_p \cdot \Delta\sigma_q)$$

其中 $(\mu_p, \sigma_p)$ 是先验参数，$(\Delta\mu_q, \Delta\sigma_q)$ 是编码器对先验的修正。优势：
- 先验与后验共享参数，减少 KL 项的优化难度
- 编码器只需学习"偏差"，优化更稳定

### 2.4 谱正则化（Spectral Regularization）

深层 VAE 训练不稳定的根源：KL 散度无界，梯度可能爆炸。谱正则化约束网络各层权重矩阵的谱范数：

$$\mathcal{L}_{SR} = \lambda \sum_i s(i)$$

其中 $s(i)$ 是第 $i$ 层权重矩阵的最大奇异值。效果：
- 防止 BN 层的缩放参数放大统计偏差
- 稳定深层 VAE 的训练（40+ 组潜在变量）
- 额外收益：轻微提升生成性能

### 2.5 KL 平衡

训练中 KL 项的梯度方差很大。引入可学习的 KL 权重，平衡不同组潜在变量的 KL 梯度：
- 对每个潜在变量组使用独立的 KL 缩放因子
- 在训练过程中动态调整以保持各组 KL 梯度量级一致

### 2.6 层次化结构

- 自底向上网络提取特征 → 共享自顶向下网络推断各组潜在变量 → 生成模型
- 潜在变量从 $8 \times 8$ 逐步上采样到 $128 \times 128$
- 大图像模型：36 组潜在变量，每组 2 个残差单元

## 三、实验结果

### 主要密度估计结果

| 数据集 | NVAE | 前SOTA (非自回归) | 自回归模型 |
|--------|------|-------------------|-----------|
| MNIST (动态二值) | **77.04** | 77.98 (MAE) | 78.00 (PixelVAE++) |
| CIFAR-10 | **2.91 bpd** | 2.98 (Flow++) | 2.85 (PixelSNAIL) |
| CelebA 64 | **3.05 bpd** | 3.16 (MaCow) | 3.05 (PixelSNAIL) |
| CelebA HQ 256×256 | **0.69 bpd** | — | 0.61 (MaCow) |

- CIFAR-10：将非自回归 SOTA 从 2.98 推进到 2.91 bpd
- **首次**成功将 VAE 应用于 256×256 自然图像

### 消融实验（CIFAR-10, 小模型）

| 组件 | NLL (bpd) |
|------|-----------|
| WN + ELU (L=40) | 3.31 |
| BN + ELU (L=40) | 3.22 |
| BN + Swish (L=40) | **3.16** |
| 无谱正则化 | 3.18 |
| 无 SE | 3.22 |
| 无残差分布 | 3.19 |

## 四、个人评价

### 影响
1. 证明网络架构设计对 VAE 性能至关重要，与统计方法（如更紧的变分界）是正交且互补的方向
2. 首次将 VAE 扩展到 256×256 自然图像生成，打破了"VAE 只能生成模糊图像"的固有印象
3. 重新确立了 BN 在 VAE 中的价值（此前被认为有害），并提供了合理的解释和解决方案
4. 残差参数化简化了层次化 VAE 的 KL 优化，影响了后续 VAE 架构设计
5. 谱正则化为深层 VAE 的训练稳定性提供了通用方案

### 局限
1. 尽管性能大幅提升，与自回归模型仍有差距（如 CIFAR-10 上 2.91 vs 2.85 bpd）
2. 训练计算量大（36 组潜在变量 + 深层网络），内存消耗显著
3. 采样时 BN 的处理需要额外调整（类似 BigGAN 的 running statistics 重校准）
4. 谱正则化的超参数 $\lambda$ 需要调节
5. 架构改进主要针对图像，对其他数据类型的适用性未验证
