# Generative Adversarial Nets (Goodfellow et al., 2014)

**arxiv**: 1406.2661 | **GAN 开山之作**

## 一、理论推导

### 1. 对抗博弈框架

定义两个模型：
- **生成器** $G(z; \theta_g)$：将噪声 $z \sim p_z(z)$ 映射到数据空间
- **判别器** $D(x; \theta_d)$：输出 $x$ 来自真实数据而非生成数据的概率

两者进行极小极大博弈：

$$\min_G \max_D V(D, G) = \mathbb{E}_{x \sim p_{data}(x)}[\log D(x)] + \mathbb{E}_{z \sim p_z(z)}[\log(1 - D(G(z)))]$$

### 2. 最优判别器

**命题 1**：对任意固定的 $G$，最优判别器为：

$$D_G^*(x) = \frac{p_{data}(x)}{p_{data}(x) + p_g(x)}$$

**证明**：将 $V(G,D)$ 重写为关于 $x$ 的积分：

$$V(G, D) = \int_x [p_{data}(x) \log D(x) + p_g(x) \log(1 - D(x))] dx$$

对每个 $x$，函数 $a \log y + b \log(1-y)$ 在 $y = \frac{a}{a+b}$ 处取最大值。令 $a = p_{data}(x)$，$b = p_g(x)$ 即得。

### 3. 全局最优性

**定理 1**：$C(G) = \max_D V(G, D)$ 的全局最小值当且仅当 $p_g = p_{data}$ 时达到，此时 $C(G) = -\log 4$。

**证明**：将最优判别器代入目标函数：

$$C(G) = \mathbb{E}_{x \sim p_{data}}\left[\log \frac{p_{data}(x)}{p_{data}(x) + p_g(x)}\right] + \mathbb{E}_{x \sim p_g}\left[\log \frac{p_g(x)}{p_{data}(x) + p_g(x)}\right]$$

减去 $-\log 4$ 后得到：

$$C(G) = -\log 4 + KL\left(p_{data} \left\| \frac{p_{data}+p_g}{2}\right.\right) + KL\left(p_g \left\| \frac{p_{data}+p_g}{2}\right.\right)$$

即：

$$C(G) = -\log 4 + 2 \cdot \text{JSD}(p_{data} \| p_g)$$

Jensen-Shannon 散度非负且当两分布相等时为零，因此全局最小值 $-\log 4$ 仅在 $p_g = p_{data}$ 时达到。

### 4. 收敛性

**命题 2**：若 $G$ 和 $D$ 有足够容量，且每步判别器达到最优，则 $p_g$ 收敛到 $p_{data}$。

关键：$\sup_D U(p_g, D)$ 关于 $p_g$ 是凸函数，且全局最优唯一，因此梯度下降收敛。

### 5. 训练技巧

- 实践中不训练 $D$ 到最优（计算代价高且过拟合），而是交替训练 $k$ 步 $D$ + 1 步 $G$
- **避免梯度消失**：早期 $G$ 很差时，$D$ 能轻易判别导致 $\log(1-D(G(z)))$ 饱和。改用 $\max_G \mathbb{E}[\log D(G(z))]$ 替代 $\min_G \mathbb{E}[\log(1-D(G(z)))]$，不动点相同但梯度更强

## 二、最小实验复现资源

| 项目 | 要求 |
|------|------|
| **数据集** | MNIST, TFD, CIFAR-10 |
| **生成器** | MLP (ReLU + sigmoid 激活) |
| **判别器** | MLP (maxout 激活 + dropout) |
| **训练** | Minibatch SGD + momentum, k=1 |
| **评估** | Gaussian Parzen window 估计 log-likelihood |
| **硬件** | 单 GPU (2014年水平), <1 小时 |
| **总估算** | **单 GPU, ~1-2 小时, <4GB 内存** |

复现代码约 150 行 Theano/PyTorch。

## 三、重要 Idea

1. **对抗训练框架**：将生成模型训练转化为两人零和博弈。生成器和判别器互相对抗，各自提升能力。这个框架的优雅之处在于不需要显式定义概率分布，只需要采样能力。

2. **目标函数等价于 JSD 最小化**：看似复杂的对抗目标，本质上等价于最小化真实分布与生成分布之间的 Jensen-Shannon 散度。这给出了清晰的理论保证。

3. **无需 MCMC**：与当时主流的 RBM/DBN/DBM 方法不同，GAN 完全不需要马尔可夫链进行采样或推断。生成只需一次前向传播。这是计算上的巨大优势。

4. **训练技巧 —— 非饱和目标**：$\log(1-D(G(z)))$ 在训练初期梯度消失，替换为 $\log D(G(z))$ 是实践中的关键修改。这个看似微小的改动对训练稳定性影响巨大。

5. **模式崩塌 (Helvetica scenario)**：论文已经预见性地指出了 GAN 的核心问题 —— 生成器可能将多个 $z$ 映射到同一个 $x$，导致多样性不足。这成为后来大量 GAN 变体致力于解决的问题。

## 评价

这篇仅 9 页的论文提出了深度生成模型中最具影响力的框架之一。理论分析简洁优美，将对抗训练与 JSD 建立了严格联系。但论文也留下了许多未解决的问题：训练不稳定、模式崩塌、评估困难（Parzen window 估计粗糙）、缺少显式似然。这些问题催生了后续数千篇论文。与 VAE（同期）相比，GAN 生成质量通常更高但训练更困难、更不稳定，两者各有优劣。
