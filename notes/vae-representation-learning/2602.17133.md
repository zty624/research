## 1. 理论推导

### 核心观点: 量化 = 结构化扰动

标准 VQ-VAE 中, 量化操作将隐向量 z 映射到最近码本向量 q(z), 引入量化误差 eps = q(z) - z。VP-VAE 的核心洞察: 从解码器视角看, 量化本质上是在隐空间中注入一个有界的、局部的扰动(即量化误差)。因此, 训练时无需执行离散量化, 只需让解码器学会容忍与量化误差匹配的扰动即可。

### 与标准 VAE 重参数化的区别

标准 VAE 的重参数化技巧: z = mu + sigma * eps, eps ~ N(0, I), 采样分布由编码器参数化的高斯分布决定, KL 正则约束后验接近先验。VP-VAE 完全不同: 它不是从参数化分布中采样, 而是对编码器输出 z 施加自适应扰动 z_tilde = T(z; S), 其中 T 是基于 Metropolis-Hastings 采样的扰动算子, S 是近期隐向量的 FIFO 队列。VP-VAE 没有 KL 项, 也不假设隐变量服从高斯先验, 而是通过非参数密度估计让扰动与经验隐空间密度一致。

### 低维瓶颈设计

为解决高维空间密度估计的困难, VP-VAE 引入可学习的降投影/升投影:

- z = P_down(h_t) in R^d  (降投影到低维, d <= 16)
- h_tilde = P_up(z_tilde)  (升投影回原始空间)

扰动和量化均在 d 维压缩空间中进行, 解码器从升投影后的特征重建输入。

### 扰动尺度的自适应估计 (Scale Alignment)

维护 FIFO 队列 S 存储近期隐向量。对于目标码本大小 K, 定义局部量化半径:

M = ceil(|S| / K)
R(z) = eta * D_M(z | S)

其中 D_M(z|S) 是 z 到 S 中第 M 近邻的欧氏距离, eta > 0 是超参数。直觉: 若 K 个码本向量将 |S| 个样本等概率划分, 每个区域约含 M 个样本, 第 M 近邻距离即为局部 Voronoi 胞半径的自适应估计。

### 分布一致性: Metropolis-Hastings 采样

用 kNN 估计目标密度: pi(z) proportional to 1 / D_k(z | S)^d

**提议分布**: 从以 z 为中心、半径 R(z) 的 d 维球均匀采样:
z' = z + u, u ~ Unif(B(0, R(z)))

**接受概率** (由 MH 算法推导):
alpha(z, z') = min(1, [D_k(z) * D_M(z)]^d / [D_k(z') * D_M(z')]^d)

附加支撑条件: 若 ||z - z'|| > R(z'), 则 alpha = 0。

**最终扰动**:
z_tilde = z' (概率 alpha), 或 z (概率 1 - alpha)

MH 的关键性质: 若 z ~ pi, 则 z_tilde ~ pi 亦成立, 保证扰动后仍在高密度区域。

### 优化目标

L = L_rec(x, x_hat) + L_norm

L_norm = lambda_1 * ||mu_batch||^2 + lambda_2 * ||sigma_batch^2 - 1||^2

L_norm 对隐向量做零均值单位方差正则, 防止隐空间坍缩/爆炸, 并使 kNN 距离估计更可靠。

### 训练后码本生成

训练时不使用码本。训练收敛后, 在训练集上运行编码器收集隐向量, 用 K-Means++ 聚类得到 K 个质心构成码本 Q。推理时用最近邻量化: q(z) = argmin_{c in Q} ||z - c||^2。

### FSP (Finite Scalar Perturbation) 简化变体

当隐变量近似均匀分布时, VP-VAE 可大幅简化为 FSP:

1. 用 CDF-like 激活 (sigmoid/tanh/正态CDF) 将预激活 a 映射到 [0,1]^d: z = g(a)
2. 扰动简化为每维独立均匀噪声: z'_i = z_i + u_i, u_i ~ U(-eta/(2L_i), eta/(2L_i))
3. 若 z' 超出 [0,1]^d 则拒绝(保持 z 不变)
4. 量化到 Lloyd-Max 质心: z_tilde_i = (l_i + 1/2) / L_i, 其中 l_i = clip(floor(L_i * z_i), 0, L_i - 1)
5. 训练时以 1/2 概率施加扰动, 1/2 概率施加 STE 质心量化

FSP 的理论优势: 对于均匀分布, Lloyd-Max 质心 (区间中点) 是最优标量量化重建点, 优于 FSQ 的取整方式。

## 2. 最小实验复现资源

- **数据集**:
  - 图像: COCO 2017 训练集, COCO 验证集 + ImageNet 验证集评估 (图像 resize 短边 256, center-crop 256x256)
  - 音频: LibriSpeech train-clean-460 + train-other-500 训练, LibriSpeech test-clean/test-other + Common Voice v18.0 English 评估, 16kHz 重采样
- **模型架构**:
  - 图像: VQGAN 架构 (Esser et al., 2021), 编码器 8x 下采样, 32x32 token 网格
  - 音频: SQCodec 架构 (Zhai et al., 2025), 约 166.67 Hz 帧率
- **硬件**: 2x NVIDIA RTX 4090
- **训练时间**: 所有模型训练 50 epochs
- **优化**: 使用原论文相同的优化器和超参数
- **损失函数**:
  - 图像: L1 + LPIPS 感知损失
  - 音频: L1 时域重建 + 多分辨率 STFT 损失 + Whisper 特征感知损失
- **不使用 GAN/Discriminator**, 以公平比较量化机制
- **码本大小**: K in {256, 1024, 4096, 16384}
- **VP-VAE 隐空间维度**: 与 FSQ 配置一致 (K=256: d=3, K=1024: d=4, K=4096: d=5, K=16384: d=5)

## 3. 重要idea

### (a) 向量扰动替代离散量化: 从"先量化再训练"到"先训练再量化"

这是本文最根本的思想转变。传统 VQ-VAE 在训练中同时优化编码器/解码器和码本, 形成耦合: 编码器必须跟踪移动的码本目标, 码本必须适应变化的编码器分布。VP-VAE 彻底解耦: 训练阶段只有编码器-解码器, 通过扰动模拟量化效果; 码本在训练收敛后才通过 K-Means 离线生成。这从根本上消除了码本坍缩的根源。

### (b) 自适应扰动: 尺度对齐 + 分布一致性

两个关键设计使扰动有效逼近量化误差:
- **尺度对齐**: 通过 kNN 估计局部 Voronoi 胞半径, 扰动幅度自动适应局部密度和目标码本大小, 不需要手动调噪声强度。
- **分布一致性**: MH 采样保证扰动后的隐向量仍在原始分布的高密度区域内, 避免"无效隐状态"浪费解码器容量。消融实验表明, 移除 MH (即总是接受提议, 退化为均匀噪声) 导致 CVU 从 0.81 降到 0.75。

### (c) FSP: 均匀分布假设下的理论最优简化

当隐变量近似均匀时, MH 密度估计退化为简单的边界检查, 扰动退化为每维独立均匀噪声, 量化使用 Lloyd-Max 质心而非 FSQ 的取整。这提供了 FSQ 的理论解释和实际改进: FSQ 取整到网格边界导致输出分布偏向边界, 而 FSP 量化到区间中点保持输出近似均匀。实验中 FSP 在所有条件下一致优于 FSQ, 且计算复杂度相当。

### (d) 跨模态稳定性

SimVQ 在图像上表现优秀但在音频上严重坍缩 (CVU 低至 10^-4), FSQ 在音频上稳定但在图像上重建质量不如 VP-VAE/FSP。VP-VAE 和 FSP 在两个模态上都保持竞争力, 说明解耦训练范式不依赖模态特定启发式, 具有更好的泛化性。

### (e) 局限性

VP-VAE 的 kNN 搜索在队列 S 增大时带来额外训练计算开销 (虽然通过降维和随机子采样缓解)。当前实验限于自编码器 tokenization, 与自回归或扩散 token 生成器的整合是未来方向。
