# Equivariant Contrastive Learning (Dangovski et al., 2021)

**arxiv**: 2111.00899 | **等变性, 自监督学习, 对比学习** | MIT / FAIR | ICLR 2022

---

## 一、核心问题

当前自监督学习（SSL）方法（如 SimCLR、BYOL）的核心原则是**不变性**（invariance）：对输入施加增强后，要求表征保持不变。但不变性只是**等变性**（equivariance）的特例——等变性要求表征按照输入变换的方式相应变换。

核心问题：能否利用非平凡的等变性作为额外的预训练目标来提升表征质量？

## 二、方法

### 1. 不变性 vs. 等变性

- **不变性**：$f(T(x)) = f(x)$，增强后表征不变
- **等变性**：$f(T(x)) = T'(f(x))$，表征按可预测方式变换

等变性是不变性的推广。当 $T' = \text{id}$ 时退化为不变性。

### 2. E-SSL 框架

在标准 SSL 方法基础上增加一个简单的等变预测头：

$$\hat{t} = g(f(x), f(T(x)))$$

其中 $g$ 是一个小型预测网络，目标是预测对 $x$ 施加的变换 $T$ 的参数。

**训练目标**：标准 SSL 损失 + 等变预测损失

$$\mathcal{L} = \mathcal{L}_{\text{SSL}} + \lambda \mathcal{L}_{\text{eq}}$$

### 3. 具体实现

以 SimCLR 为基础（E-SimCLR）：
- 在投影头之后添加等变预测头
- 对旋转角度、颜色抖动等连续变换预测变换参数
- 离散变换（如水平翻转）用交叉熵损失
- 连续变换用 MSE 损失

关键设计选择：
- **部分等变 + 部分不变**：对某些变换保持不变性（如裁剪、高斯模糊），对另一些施加等变性（如旋转、颜色）
- 等变预测头的梯度不会干扰主干表征学习

### 4. 理论分析

从信息论角度：
- 不变性目标可能**丢失**与变换相关的信息
- 等变性目标**保留**变换信息，同时仍学习语义表征
- 两者结合可以学到更丰富的表征

## 三、实验结果

### 1. ImageNet Linear Probe

| 方法 | Top-1 Acc. |
|------|-----------|
| SimCLR | 69.3% |
| E-SimCLR | **72.5%** |
| BYOL | 74.3% |
| E-BYOL | **75.6%** |

等变目标在 SimCLR 上带来 +3.2% 的提升，在 BYOL 上带来 +1.3% 的提升。

### 2. 下游任务迁移

在 CIFAR-10、CIFAR-100、Food-101 等 12 个数据集上，E-SSL 一致优于基线。

### 3. 光子学回归任务

在光子学科学回归问题中，E-SSL 显著优于标准 SSL，展示了等变性在非视觉任务中的价值。

### 4. 消融实验

- 旋转等变性贡献最大
- 同时使用多种等变性优于单一等变性
- 等变损失权重 $\lambda$ 在 0.1-1.0 范围内鲁棒

## 四、个人评价

**优点**：
1. 概念优雅——将 SSL 从"不变性"推广到"等变性"是自然且深刻的
2. 实现简洁——仅在投影头后加一个小预测网络，几乎不增加计算开销
3. 通用性强——可应用于 SimCLR、BYOL、MoCo 等多种 SSL 框架
4. 跨域验证——在光子学任务上的成功展示了通用性

**不足**：
1. 等变变换的选择仍需人工设计，不同任务需要不同的等变性
2. 对 SimCLR 提升大（+3.2%），但对已经很强的 BYOL 提升较小（+1.3%），暗示等变信息可能已被隐式学习
3. 缺少与专门设计等变架构（如群等变 CNN）的对比
4. 理论分析偏直觉，缺少对"何时等变性有帮助"的形式化刻画

**影响**：E-SSL 开辟了 SSL 研究的新方向——从纯粹的不变性到更精细的等变性建模。后续的 EqSim、Contrastive Equivariant Learning 等工作都受此启发。核心洞察"不变性是等变性的特例"对 SSL 目标函数的设计具有持久影响。
