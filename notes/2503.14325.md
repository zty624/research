## 一、理论推导

### 1. 整体框架

输入视频 $x \in \mathbb{R}^{(T+1) \times H \times W \times 3}$，编码为隐变量 $z \in \mathbb{R}^{(T'+1) \times H' \times W' \times d}$，压缩比为 $c_t = T/T'$ (时间), $c_s = H/H' = W/W'$ (空间)。

流水线：Patchify → Encoder → Channel Compression Bottleneck → Latent $z$

### 2. 小波变换 (Wavelet Transform)

用 3D Haar DWT 将视频分解为 8 个子带：
- 1 个低频子带 $V_{lll}$：长期平均信息
- 7 个高频子带：局部细节和变化

对图像用 2D Haar DWT 分解为 4 个子带。IDWT 可完美重建。

优势：丰富特征空间，扩展 CNN 感受野，spatiotemporal 维度减半而通道增加。

### 3. Patchify

- 第一帧（图像）和后续帧分开处理
- 低频和高频分量分别 patch + 线性投影
- 非重叠 patch (2×4×4)，减少计算
- **不对 patch 做 normalization**（发现 LayerNorm 导致 block artifacts）

### 4. NAF 模块 (Neighborhood-Aware Feedforward)

轻量级骨干模块：
- 3×3×3 depthwise conv → 聚合局部上下文
- Feedforward layer → 特征变换
- 残差连接

编码器 $\xi = \{\xi_l, \xi_h, \xi_f\}$：分别处理低频、高频、融合特征，各 2/2/4 层 ResNAF。

### 5. 通道压缩瓶颈 (Compressed Sensing)

**核心创新**：用压缩感知替代简单的两层线性 AE 做通道压缩。

下采样：感知矩阵 $\Phi \in \mathbb{R}^{d \times D}$，$z = \Phi p$（$D \to d$）

上采样（恢复）：基于 ISTA-Net+ 的迭代优化：

$$r^{(k)} = p^{(k-1)} - \rho^{(k)} \Phi^\top(\Phi p^{(k-1)} - z)$$
$$p^{(k)} = r^{(k)} + F_e^{(k)}(\text{soft}(F^{(k)}(r^{(k)}), \theta^{(k)}))$$

其中 $\text{soft}(x, \theta) = \text{sgn}(x) \cdot \max(|x| - \theta, 0)$ 为软阈值收缩。

参数 $\Theta = \{\rho^{(k)}, \theta^{(k)}, F^{(k)}, F_e^{(k)}\}_{k=1}^K$，$K=2$ 次迭代。

### 6. 因果性

时间维度的 depthwise conv 使用因果 padding，确保每帧只与之前帧交互。支持：
- 图像建模（独立于邻居帧）
- 因果隐空间下游模型
- 高效分块推理（缓存前块帧）

## 二、最小实验复现资源

| 项目 | 要求 |
|------|------|
| **数据集** | Panda-70M, OpenWebVid (视频预训练), ImageNet, COCO |
| **模型** | ResNAF backbone, D=512, d=4或16, K=2 ISTA 迭代 |
| **压缩比** | 4×8×8 (时间×空间高×空间宽) |

| **总估算** | **8 A100, ~2 天** |

## 三、重要 Idea

1. **非重叠 Patch + 频域处理**：将 3D 视频通过 Haar 小波变换转到频域后再 patchify，不同于传统 ViT 在 RGB 空间 patch。频域表示天然具有多尺度信息，且 patch 操作更高效。

2. **压缩感知做通道压缩**：这是最重要的创新。传统方法用两层线性层做通道降维/升维，信息损失大。LeanVAE 用 ISTA-Net+ 的迭代优化框架，将压缩感知的数学原理（稀疏恢复）嵌入神经网络，实现更高质量的通道压缩和恢复。

3. **NAF 模块极致轻量**：depthwise conv + FFN + 残差，比 3D 卷积 + attention 的传统方案轻得多。50× FLOPs 减少的同时保持竞争力重建质量。

4. **不做 Patch Normalization**：发现 LayerNorm 应用于视觉 patch 会导致 block artifacts，这在 ViT 中被忽视但对低层视觉任务至关重要。

5. **图像-视频联合建模**：首帧单独处理，支持图像和视频的统一编码器。

## 评价

LeanVAE 是视频 VAE 领域的工程突破。将小波变换和压缩感知两个信号处理经典工具引入 Video VAE，并用精心设计的轻量架构实现极致效率。压缩感知做通道瓶颈是最大亮点 —— 理论上优雅（稀疏恢复），实践上有效。局限：论文偏工程导向，理论分析较少；ISTAnet+ 的迭代增加了一些推理开销；与其他 Video VAE 的公平比较受训练数据差异影响。但作为面向实际部署的 Video VAE，其效率-质量权衡是当前最佳的。
