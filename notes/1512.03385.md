# Deep Residual Learning for Image Recognition (He et al., 2016)

**arxiv**: 1512.03385 | **ResNet 开山之作** | CVPR 2016

---

## 一、核心问题：深层网络退化

深度网络理论上应该更好——浅网络的解空间是深网络的子集，深网络至少可以学成"浅网络 + 恒等映射"。但实际实验发现一个反直觉的现象：**更深的网络反而训练误差更高**。

这不是过拟合（过拟合应该是训练误差低、测试误差高），而是一个**优化问题**——更深的网络更难优化。即使 BN 缓解了梯度消失，深层网络仍然比浅层网络收敛到更差的解。

这个 degradation problem 是 ResNet 要解决的核心问题。

## 二、关键创新：残差学习

### 核心思想

假设期望的映射为 $\mathcal{H}(x)$，不让网络直接拟合 $\mathcal{H}(x)$，而是拟合残差：

$$\mathcal{F}(x) := \mathcal{H}(x) - x$$

即网络只需学习：

$$\mathcal{H}(x) = \mathcal{F}(x) + x$$

### 为什么这样更好？

1. **恒等映射是最优时的下界**：如果恒等映射已经是最优的，那么残差学习只需将 $\mathcal{F}(x)$ 推向 0，这比让一堆非线性层学恒等映射容易得多
2. **梯度直通**：反向传播时，梯度可以直接通过 shortcut 连接传回，不经过任何权重层，缓解了深层梯度消失问题

直觉上：直接学恒等映射要求非线性层精确复现输入，这对 ReLU 网络很困难；学 $\mathcal{F}(x) = 0$ 只需权重趋于零即可。

## 三、方法细节

### 1. 残差块

$$y = \mathcal{F}(x, \{W_i\}) + x$$

其中 $\mathcal{F}(x, \{W_i\})$ 是要学习的残差映射，$+x$ 是 shortcut/identity mapping。

- shortcut 连接不引入额外参数，不增加计算量
- 要求 $\mathcal{F}$ 和 $x$ 维度相同

### 2. 维度不匹配时的 Shortcut Projection

当输入输出维度不同时（如通道数变化或下采样），用 $1 \times 1$ 卷积做投影：

$$y = \mathcal{F}(x, \{W_i\}) + W_s x$$

论文实验了两种策略：
- **Option A**：恒等 shortcut + zero-padding 补维度（免费）
- **Option B**：用 $1 \times 1$ 卷积投影（增加少量参数）

实验表明 Option B 略好，但差异不大，说明投影不是关键，shortcut 的存在本身才是关键。

### 3. Bottleneck 设计

对于更深的网络（50/101/152 层），使用 bottleneck 残差块：

$$1 \times 1 \ (\text{降维}) \rightarrow 3 \times 3 \ (\text{卷积}) \rightarrow 1 \times 1 \ (\text{升维})$$

例如 256-d → 64-d → 64-d → 256-d。设计思路：
- 两个 $1 \times 1$ 卷积分别负责降维和升维
- 中间的 $3 \times 3$ 卷积在低维空间操作，计算量大幅减少
- 整体计算量与两层 $3 \times 3$ 的非 bottleneck 块相当

### 4. 网络架构

| 网络深度 | 层数 | 普通/残差 | Top-5 Error |
|---------|------|----------|-------------|
| 18 | 18 | 10.02 / -- | -- |
| 34 | 34 | 10.02 / 8.73 | 有改善 |
| 50 | 50 | bottleneck | 6.71 |
| 101 | 101 | bottleneck | 5.69 |
| 152 | 152 | bottleneck | **4.49** |

关键观察：普通网络在 34 层就出现退化（比 18 层差），而残差网络 34 层比 18 层好，完全消除了退化问题。

## 四、实验结果

### ImageNet 分类

- **152 层 ResNet** 达到 **3.57% top-5 error**（ensemble），获得 ILSVRC 2015 冠军
- 单模型 152 层 ResNet top-5 error 为 4.49%，已优于 VGGNet/GoogLeNet 的 ensemble
- 比 VGG-16 深 8 倍，但复杂度反而更低（35.6 亿 FLOPs vs VGG 的 196 亿 FLOPs）

### CIFAR-10

- 成功训练 **1202 层** 的网络（虽然测试误差略高于 110 层，说明可能存在过拟合）
- 110 层 ResNet 达到约 6.43% 测试误差
- 消除了深层退化：普通网络 34 层就开始退化，ResNet 1202 层仍能正常训练

### COCO 检测与分割

- ResNet 作为 backbone 获得 COCO 2015 检测冠军
- 比 VGG backbone 提升 ~28% 的 mAP

## 五、为什么残差有效

### 1. 恒等映射容易学习

如果最优映射接近恒等映射，$\mathcal{F}(x) = 0$ 很容易学到（权重趋向零即可），而让一堆非线性层精确复现输入很难。

### 2. 梯度可以直接传播

反向传播时：

$$\frac{\partial \mathcal{L}}{\partial x} = \frac{\partial \mathcal{L}}{\partial y} \left( \frac{\partial \mathcal{F}}{\partial x} + 1 \right)$$

其中 $+1$ 这一项保证梯度至少为 $\frac{\partial \mathcal{L}}{\partial y}$，不会因为经过多层而消失。即使 $\mathcal{F}$ 的梯度很小，信号仍可通过 shortcut 直接传回。

### 3. 等价于浅网络的集成

Veit et al. (2016) 后续分析表明，ResNet 的行为更像一组浅网络的 ensemble，而非单一极深网络。删除任意一个残差块，性能下降很小（不像普通网络删除一层就崩溃），说明信息有冗余路径。

## 六、影响

1. **开启极深网络时代**：从 VGG 的 19 层直接跃迁到 152 层乃至上千层，彻底改变了深度网络的深度认知
2. **计算机视觉的标准 backbone**：ResNet 成为目标检测（Faster R-CNN）、语义分割、视频理解等几乎所有 CV 任务的默认特征提取器
3. **残差连接成为标配**：Transformer 中的 Add & Norm、U-Net 中的 skip connection、DenseNet 的密集连接，都是残差思想的延伸
4. **直接启发的后续工作**：
   - **Highway Network**：门控残差，可学习的 shortcut
   - **DenseNet**：跨层密集连接，特征复用
   - **Pre-Activation ResNet** (He et al., 2016)：BN-ReLU-Conv 顺序更优
   - **ResNeXt**：分组卷积 + 残差，cardinality 维度
5. **超越视觉**：残差连接的思想扩散到 NLP（Transformer）、语音、科学计算等领域，成为深度学习的通用设计模式

## 评价

ResNet 的贡献不仅仅是"加了一条 skip connection"这么简单——它准确诊断了深层网络退化的优化问题，给出了优雅的数学表述（残差学习），并提供了全面的实验验证。论文写作清晰，从问题定义到方法到实验逻辑链完整。

最有说服力的证据是那个 plain network vs residual network 的对比图：34 层 plain 网络比 18 层差，而 34 层 ResNet 比 18 层好——这直接证明了残差学习解决了退化问题，而非仅仅是正则化效果。

如果要说不足，论文对"为什么残差有效"的理论分析比较初步，更像是直觉而非严格证明（后来的工作补上了这部分）。Bottleneck 设计中 $1 \times 1$ 卷积的比例选择也缺乏系统性分析。但这些丝毫不影响这篇论文的开创性地位。
