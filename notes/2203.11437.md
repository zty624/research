# Representation Uncertainty in Self-Supervised Learning as Variational Inference (Nakamura et al., 2022)

**arxiv**: 2203.11437 | **自监督学习, 变分推断, 不确定性, SimSiam** | Panasonic / Ritsumeikan University | ICCV 2023 Workshop

---

## 一、核心问题

自监督学习（SSL）和变分自编码器（VAE）都是无标签表征学习方法，但两者的理论联系从未被系统研究：

- **SSL**：最大化同一样本不同增强视图的表征相似性，学到确定性表征
- **VAE**：通过变分推断训练概率生成模型，学到分布表征

核心问题：
1. SSL 和 VAE 之间的理论关系是什么？
2. 能否在 SSL 框架中引入表征不确定性估计？

## 二、方法

### 1. SSL 与变分推断的理论联系

作者证明了 SimSiam 的损失函数可以重新解释为变分下界（ELBO）。

**SimSiam 损失**：
$$\mathcal{L}_{\text{SimSiam}} = -\frac{f(x_1)}{\|f(x_1)\|} \cdot \frac{\text{sg}(f(x_2))}{\|\text{sg}(f(x_2))\|}$$

其中 $f$ 是编码器，$\text{sg}$ 是 stop-gradient 操作。

**变分推断视角**：将表征 $\mathbf{z}$ 视为潜变量，定义：
- 先验 $p(\mathbf{z}) = \mathcal{N}(0, I)$
- 似然 $p(\tilde{x}|\mathbf{z})$ 由解码器定义
- 推断 $q_\phi(\mathbf{z}|x) = \mathcal{N}(\mu_\phi(x), \Sigma_\phi(x))$

SimSiam 的负余弦相似度损失等价于 ELBO 中的重建项（在特定假设下）。

### 2. VI-SimSiam

基于理论联系，提出 VI-SimSiam——在 SimSiam 框架中引入潜空间分布：

**关键改动**：
- 编码器输出**均值 $\mu$ 和方差 $\sigma^2$**，而非确定性向量
- 推断分布：$q_\phi(\mathbf{z}|x) = \mathcal{N}(\mu_\phi(x), \text{diag}(\sigma^2_\phi(x)))$
- 使用重参数化技巧采样 $\mathbf{z} \sim q_\phi(\mathbf{z}|x)$
- 在余弦相似度计算中使用采样的 $\mathbf{z}$

**损失函数**：
$$\mathcal{L}_{\text{VI-SimSiam}} = -\frac{\mathbf{z}_1}{\|\mathbf{z}_1\|} \cdot \frac{\text{sg}(\mathbf{z}_2)}{\|\text{sg}(\mathbf{z}_2)\|} + \beta \cdot D_{KL}(q_\phi(\mathbf{z}|x) \| p(\mathbf{z}))$$

其中 $\beta$ 控制 KL 正则项的权重。

### 3. 表征不确定性

VI-SimSiam 的核心价值在于能估计**表征的不确定性**：
- 方差 $\sigma^2$ 大的维度 → 对应特征不确定
- 方差 $\sigma^2$ 小的维度 → 对应特征确定

不确定性可用于：
- 检测 OOD 样本
- 主动学习（选择高不确定性样本标注）
- 下游任务的置信度估计

## 三、实验结果

### 1. 不确定性的定性分析

将估计的方差 $\sigma^2$ 可视化为热力图：
- 简单/常见图像（如清晰的单个物体）：整体不确定性低
- 复杂/模糊图像：不确定性高
- 遮挡、模糊、低对比度区域：局部不确定性升高

### 2. 不确定性与分类准确率的关系

- 不确定性低的样本 → 下游分类准确率高
- 不确定性高的样本 → 下游分类准确率低
- 验证了不确定性估计的校准性

### 3. Linear Probe 性能

| 方法 | CIFAR-10 | CIFAR-100 |
|------|----------|-----------|
| SimSiam | 91.2% | 67.3% |
| VI-SimSiam | 90.8% | 66.9% |

VI-SimSiam 在分类精度上与 SimSiam 基本持平，但额外提供了不确定性估计。

### 4. OOD 检测

利用方差作为 OOD 分数，在 SVHN vs CIFAR-10 的 OOD 检测上展示了初步效果。

## 四、个人评价

**优点**：
1. 理论贡献——首次建立 SSL（SimSiam）与变分推断的严格联系，深化了对 SSL 目标函数的概率解释
2. 实现简洁——仅需将确定性编码器改为输出均值和方差，加入重参数化和 KL 项
3. 不确定性估计的实用性——零额外计算开销（编码时即获得），可直接用于 OOD 检测和主动学习
4. 定性实验直观——不确定性热力图与图像内容高度吻合

**不足**：
1. 分类性能略有下降（~0.4%），虽小但说明引入分布假设有代价
2. 理论推导中对 SimSiam stop-gradient 的处理不够完整——stop-gradient 在变分框架中没有自然的对应
3. OOD 检测实验规模偏小，缺少与专门 OOD 方法的系统对比
4. 仅基于 SimSiam，未扩展到 SimCLR、BYOL 等更主流方法
5. $\beta$ 的选择对结果敏感，缺少系统性的调参指南

**影响**：这篇工作在 SSL 和概率建模之间建立了桥梁，启发了后续关于 SSL 不确定性估计的研究。其核心洞察——SSL 的对比损失可以从变分推断角度理解——为 SSL 提供了新的理论基础，也为 VAE 社区提供了新的视角。
