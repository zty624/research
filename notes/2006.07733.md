# Bootstrap Your Own Latent: A New Approach to Self-Supervised Learning (Grill et al., 2020)

**arxiv**: 2006.07733 | **关键词**: BYOL, 自监督学习, 无负样本, 动量网络 | DeepMind / Imperial College | NeurIPS 2020

---

## 一、核心问题

当前最先进的自监督视觉表示学习方法（MoCo、SimCLR 等）都依赖负样本对来避免表示坍缩。但负样本对需要仔细处理（大 batch、memory bank、挖掘策略），且性能对数据增强选择非常敏感。能否不使用负样本，仅通过正样本对就学到好的表示？

## 二、方法

### 2.1 核心思想

BYOL 使用两个网络互相学习：在线网络（online）预测目标网络（target）的表示。目标网络的参数是在线网络参数的指数移动平均，无需负样本即可避免坍缩。

### 2.2 架构

**在线网络**（参数 $\theta$）：编码器 $f_\theta$ + 投影器 $g_\theta$ + 预测器 $q_\theta$

**目标网络**（参数 $\xi$）：编码器 $f_\xi$ + 投影器 $g_\xi$（无预测器）

给定图像 $x$ 的两个增强视图 $v$ 和 $v'$：

$$y_\theta = f_\theta(v), \quad z_\theta = g_\theta(y_\theta), \quad q_\theta(z_\theta) \text{ （预测）}$$
$$y'_\xi = f_\xi(v'), \quad z'_\xi = g_\xi(y'_\xi) \text{ （目标）}$$

### 2.3 损失函数

L2 归一化后的 MSE 损失：

$$\mathcal{L}_{\theta,\xi} = \|\bar{q}_\theta(z_\theta) - \bar{z}'_\xi\|_2^2 = 2 - 2 \cdot \frac{\langle q_\theta(z_\theta), z'_\xi \rangle}{\|q_\theta(z_\theta)\|_2 \cdot \|z'_\xi\|_2}$$

对称化：交换 $v$ 和 $v'$ 的角色计算 $\tilde{\mathcal{L}}_{\theta,\xi}$，总损失 $\mathcal{L}^{\text{BYOL}}_{\theta,\xi} = \mathcal{L}_{\theta,\xi} + \tilde{\mathcal{L}}_{\theta,\xi}$。

仅对 $\theta$ 求梯度，$\xi$ 通过 stop-gradient 不接收梯度。

### 2.4 目标网络更新

$$\xi \leftarrow \tau \xi + (1 - \tau)\theta$$

$\tau$ 从 $\tau_{\text{base}} = 0.996$ 逐渐增加到 1（余弦调度）。

### 2.5 为何不坍缩？——条件方差视角

假设预测器最优 $q^*(z_\theta) = \mathbb{E}[z'_\xi | z_\theta]$，则 BYOL 的梯度等价于：

$$\nabla_\theta \mathcal{L} = \nabla_\theta \sum_i \mathbb{E}[\text{Var}(z'_{\xi,i} | z_\theta)]$$

即最小化给定在线投影后目标投影的条件方差。丢弃在线投影中的信息无法降低条件方差，因此在线网络被鼓励保留尽可能多的信息。目标网络通过 EMA 更新，将在线网络捕获的变异性逐步传播到目标网络。

### 2.6 关键设计：预测器 + 动量目标网络

消融实验表明：
- 无预测器 + 无目标网络（即 Mean Teacher 无监督版）：坍缩
- 有预测器 + 无目标网络：若预测器近似最优，不坍缩（52.5% 线性准确率）
- 有预测器 + 有目标网络（BYOL）：72.5%

预测器使在线和目标分支非对称，阻止坍缩；动量目标网络确保预测器始终近似最优。

## 三、实验结果

**ImageNet 线性评估（ResNet-50）**：

| 方法 | Top-1 |
|------|-------|
| MoCo v2 | 71.1% |
| SimCLR | 69.3% |
| InfoMin | 73.0% |
| **BYOL** | **74.3%** |

**更大架构**：ResNet-200(2x) 达 79.6%，ResNet-50(4x) 达 78.6%，接近监督基线 78.9%。

**半监督**：1% 标签 53.2%，10% 标签 68.8%，均优于 SimCLR。

**鲁棒性**：
- Batch size 256→4096，BYOL 几乎无性能下降，SimCLR 下降明显
- 移除颜色增强：BYOL 下降 9.1%，SimCLR 下降 22.2%
- 仅用随机裁剪：BYOL 59.4%，SimCLR 40.3%

## 四、个人评价

### 影响

BYOL 是自监督学习的范式转换之作，首次证明不使用负样本也能达到 SOTA，打破了"对比学习必须依赖负样本"的固有认知。其"自举"思想启发了后续一系列非对比方法（如 SimSiam、Barlow Twins、VICReg）。BYOL 对数据增强和 batch size 的鲁棒性也使其更易应用于实际场景。

### 局限

1. 理论上对为何不坍缩的解释仍不完全，条件方差分析依赖于预测器最优假设
2. 训练初期 batch normalization 可能隐式提供负样本信息，引发争议（后续研究显示用 Group Norm 替换 BN 后 BYOL 仍可工作）
3. 仍依赖精心设计的图像增强，推广到其他模态需重新设计增强策略
4. 训练需要 1000 epochs，计算开销较大
