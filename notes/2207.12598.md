# Classifier-Free Diffusion Guidance (Ho & Salimans, 2022)

**arxiv**: 2207.12598 | **Classifier-Free Guidance 开山之作** | NeurIPS 2022 Workshop (长版)

---

## 一、核心动机

扩散模型已经能生成高质量图像，但存在**样本质量 vs 多样性的权衡问题**。在 GAN 中，可以通过 truncation（截断噪声输入）来控制这个权衡；在 flow 模型中，可以降低采样温度。但扩散模型中，**简单的截断/降温策略无效**——缩放 score 或减少噪声方差只会生成模糊的低质量样本。

Dhariwal & Nichol (2021) 提出了 **Classifier Guidance**：用额外训练的图像分类器的梯度来引导采样，通过调节梯度强度来权衡 IS/FID。但这有几个问题：
1. 需要额外训练一个在噪声数据上的分类器（不能用预训练分类器替代）
2. 引导采样过程类似于对分类器的**对抗攻击**，令人怀疑 FID/IS 提升是否只是因为"骗过了"评估分类器
3. 增加了训练流程的复杂度

本文的核心贡献：**完全不需要分类器，仅通过条件模型和无条件模型的 score 组合就能实现同等效果的引导**——即 Classifier-Free Guidance。

## 二、方法

### Classifier Guidance 回顾

在 classifier guidance 中，扩散模型的 score 被修改为：

$$\tilde{\epsilon}_\theta(z_\lambda, c) = \epsilon_\theta(z_\lambda, c) - w \sigma_\lambda \nabla_{z_\lambda} \log p_\theta(c|z_\lambda)$$

等价于从分布 $\tilde{p}_\theta(z_\lambda|c) \propto p_\theta(z_\lambda|c) p_\theta(c|z_\lambda)^w$ 中采样。当 $w > 0$ 时，增加了分类器对正确类别高置信度数据的概率权重，从而提升 IS 但降低多样性。

### Classifier-Free Guidance 的核心思想

利用贝叶斯定理重写条件 score：

$$\nabla_{z_\lambda} \log p(z_\lambda|c) = \nabla_{z_\lambda} \log p(z_\lambda) + \nabla_{z_\lambda} \log p(c|z_\lambda)$$

由此可得：

$$\nabla_{z_\lambda} \log p(c|z_\lambda) = \nabla_{z_\lambda} \log p(z_\lambda|c) - \nabla_{z_\lambda} \log p(z_\lambda)$$

代入 classifier guidance 公式，**用条件 score 和无条件 score 的差来替代分类器梯度**：

$$\tilde{\epsilon}_\theta(z_\lambda, c) = \epsilon_\theta(z_\lambda, c) + w(\epsilon_\theta(z_\lambda, c) - \epsilon_\theta(z_\lambda))$$

$$= (1 + w)\epsilon_\theta(z_\lambda, c) - w \cdot \epsilon_\theta(z_\lambda)$$

- $w = 0$：退化为普通条件采样
- $w > 0$：增强条件信号，减小无条件似然的同时增大条件似然
- 不需要任何分类器！

### 训练：条件与无条件模型共享权重

**核心技巧**：不需要训练两个独立模型！只需要一个模型，在训练时以概率 $p_{\text{uncond}}$ 随机丢弃条件信息 $c$（替换为空/特殊 token），这样同一个网络就能同时学习条件 score 和无条件 score。

Algorithm 1（训练）：
1. 采样 $(x, c)$
2. 以概率 $p_{\text{uncond}}$ 将 $c$ 设为空
3. 正常训练去噪模型 $\epsilon_\theta(z_\lambda, \tilde{c})$

Algorithm 2（采样）：
1. 每步计算 $\epsilon_\theta(z_\lambda, c)$ 和 $\epsilon_\theta(z_\lambda)$
2. 组合：$\tilde{\epsilon} = (1+w)\epsilon_\theta(z_\lambda, c) - w \cdot \epsilon_\theta(z_\lambda)$
3. 用组合后的 score 进行反向采样

### 直觉理解

引导的本质是**降低无条件似然、提升条件似然**。无条件 score $\epsilon_\theta(z_\lambda)$ 代表"一般数据"的方向，条件 score $\epsilon_\theta(z_\lambda, c)$ 代表"特定类别数据"的方向。两者之差 $\epsilon_\theta(z_\lambda, c) - \epsilon_\theta(z_\lambda)$ 就是从"一般"走向"特定"的方向。放大这个差值就是增强类别信号的权重。

用 toy 例子（三类高斯混合）可视化：随着 guidance 强度增加，每个类别的条件分布质量向远离其他类的方向移动，集中在更小的区域——多样性降低但个体质量提升。

## 三、实验结果

### ImageNet 64x64

| Guidance $w$ | FID (↓) | IS (↑) |
|-------------|---------|--------|
| 0.0 | 5.74 / 5.19 / 4.34 | 119.3 / 112.2 / 93.0 |
| 0.1 | 3.03 / 2.93 / 2.65 | 92.8 / 88.6 / 74.9 |
| 0.3 | 2.04 / 2.1 / 2.08 | 78.9 / 77.0 / 65.6 |
| 1.0 | 1.55 / 1.62 / 1.91 | 66.1 / 64.6 / 56.1 |
| 4.0 | 1.8 / 1.8 / 2.21 | 53.7 / 52.9 / 47.6 |

（三列分别对应 $p_{\text{uncond}} = 0.1, 0.2, 0.5$）

- **FID 在 $w \approx 0.1$~$0.3$ 时最优，IS 随 $w$ 单调递增**
- 最佳 FID 1.55 超过 ADM 的 2.07，接近 CDM 的 1.48

### ImageNet 128x128（T=256）

| Guidance $w$ | FID (↓) | IS (↑) |
|-------------|---------|--------|
| 0.0 | 7.27 | 82.5 |
| 0.3 | 2.43 | 158.5 |
| 0.5 | 2.98 | 208.0 |
| 1.0 | 7.86 | 298.0 |
| 4.0 | 19.77 | 407.4 |

- **$w = 0.3$ 时 FID=2.43 超越了 classifier-guided ADM-G 的 2.97**
- **$w = 4.0$ 时在最佳 IS 截断水平上同时超越 BigGAN-deep 的 FID 和 IS**
- 这是当时 128x128 ImageNet 上的**最佳结果**

### $p_{\text{uncond}}$ 的影响

- $p_{\text{uncond}} = 0.5$ 在整个 IS/FID 前沿上**始终差于** 0.1 和 0.2
- $p_{\text{uncond}} = 0.1$ 和 0.2 表现相当
- 结论：只需要**少量训练步**用于无条件生成就能有效引导，模型大部分容量应留给条件建模

### 采样步数的影响

- $T \in \{128, 256, 1024\}$，$T=256$ 是质量与速度的甜点
- 注意：每步需要两次前向传播（条件+无条件），所以 $T=128$ 才是公平的速度对比——此时 FID 略差于 ADM-G

### 定性观察

- 强引导样本（$w \geq 3$）呈现**色彩过饱和**现象
- 增加引导强度明显减少样本多样性，增加个体保真度

## 四、个人评价

Classifier-Free Guidance 是扩散模型领域最优雅、影响最深远的贡献之一：

1. **极简实现**：训练时加一行代码（随机丢弃条件），采样时加一行公式（线性组合两个 score）。与 classifier guidance 相比，零额外训练成本
2. **消除了对分类器的依赖**：不需要在噪声数据上额外训练分类器，也不再有"对抗攻击评估分类器"的嫌疑
3. **理论洞察深刻**：将引导的本质解释为"降低无条件似然、提升条件似然"，这个理解启发了后续很多工作
4. **实际影响巨大**：CFG 是 Stable Diffusion、DALL-E 2/3、Imagen 等所有主流条件扩散模型的标配组件，guidance scale 成了用户可调节的超参数

局限性：
1. **采样速度**：每步需要两次前向传播（条件+无条件），是 classifier guidance 速度的两倍（分类器通常更小更快）。不过实践中这个开销可以接受
2. **多样性损失**：强引导不可避免地牺牲多样性，论文对此的讨论较浅
3. **超参数 $p_{\text{uncond}}$ 的选择**：虽然论文实验表明 0.1-0.2 较好，但缺乏理论指导
4. **论文定位为 proof-of-concept**：使用的是 Dhariwal & Nichol 为 classifier guidance 调优的超参数，未专门为 CFG 调参，可能还有提升空间

## 五、我们的实验验证（55_cfg.py）

在 2D 条件数据上复现 CFG 核心行为，直观验证 guidance scale 的效果。

**设置**：4 个类（四象限高斯），条件扩散模型 + condition dropout $p=0.1$，DDPM 采样 50 步。

**关键结果**：

| Guidance Scale $w$ | 与目标中心距离 | 样本标准差 | 现象 |
|-------------------|--------------|-----------|------|
| 0（无条件） | ~0.33 | 高 | 样本散布在各类中心，不聚焦于目标类 |
| 2 | ~0.13 | 中 | 样本明显集中在目标类，距离从 0.33 降至 0.13 |
| 3+ | 更近 | 低 | 极度集中，多样性显著下降 |

核心观察：
1. **Accuracy-Diversity Trade-off**：$w$ 增大时样本更忠实于目标条件（距离下降），但多样性单调下降（std 降低），与论文理论完全一致
2. **$w=0$ 的退化行为**：无条件采样不遵循任何类别，样本散布在所有类中心附近
3. **$w=2$ 是有效的甜点**：距离大幅下降的同时多样性尚可接受
4. **所有类在 $w=3$ 时均可正确生成**，验证了 CFG 对任意条件的通用性

实验代码：`landmark-experiments/55_cfg.py`，可视化包括 guidance scale 效果图、类条件生成图、accuracy-diversity trade-off 曲线。

## 六、总结

尽管论文本身篇幅不长，但 CFG 的影响力是革命性的——它让"用文本提示词控制图像生成质量"成为可能，直接催生了 Stable Diffusion 的引导采样机制。可以说，没有 CFG，就没有后来扩散模型在创意应用中的爆发。
