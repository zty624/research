# Denoising Diffusion Probabilistic Models (Ho et al., 2020)

**arxiv**: 2006.11239 | **扩散模型开山之作** | NeurIPS 2020

---

## 一、核心贡献

本文将扩散概率模型从理论框架推向实用图像生成。核心贡献是：

1. **简化训练目标**：发现扩散模型的训练等价于去噪得分匹配（denoising score matching），由此导出极简的损失函数——只需预测每步添加的噪声
2. **无需对抗训练即可达到 GAN 级别图像质量**：在 CIFAR-10 上 FID=3.17（无条件生成），LSUN 上生成 256x256 图像质量媲美 StyleGAN2
3. **建立了扩散模型与得分匹配、Langevin 动力学的理论桥梁**，将三个看似独立的研究方向统一

## 二、关键技术点

### 前向过程（Forward Process）

给定数据 $\mathbf{x}_0 \sim q(\mathbf{x}_0)$，定义固定（不可学习）的马尔可夫链，逐步添加高斯噪声：

$$q(\mathbf{x}_t | \mathbf{x}_{t-1}) = \mathcal{N}(\mathbf{x}_t; \sqrt{1 - \beta_t}\mathbf{x}_{t-1}, \beta_t \mathbf{I})$$

其中 $\beta_1, \ldots, \beta_T$ 是线性递增的方差调度（variance schedule），$T=1000$。

**重参数化技巧**：令 $\alpha_t = 1 - \beta_t$，$\bar{\alpha}_t = \prod_{s=1}^t \alpha_s$，则可以一步从 $\mathbf{x}_0$ 采样任意时刻 $\mathbf{x}_t$：

$$q(\mathbf{x}_t | \mathbf{x}_0) = \mathcal{N}(\mathbf{x}_t; \sqrt{\bar{\alpha}_t}\mathbf{x}_0, (1 - \bar{\alpha}_t)\mathbf{I})$$

$$\mathbf{x}_t = \sqrt{\bar{\alpha}_t}\mathbf{x}_0 + \sqrt{1 - \bar{\alpha}_t}\boldsymbol{\epsilon}$$

这个性质是简化训练目标的关键——训练时不需要逐步走前向过程。

### 反向过程（Reverse Process）

反向过程也是高斯马尔可夫链，参数为 $\mu_\theta$ 和 $\Sigma_\theta$：

$$p_\theta(\mathbf{x}_{t-1} | \mathbf{x}_t) = \mathcal{N}(\mathbf{x}_{t-1}; \mu_\theta(\mathbf{x}_t, t), \Sigma_\theta(\mathbf{x}_t, t))$$

本文固定 $\Sigma_\theta(\mathbf{x}_t, t) = \sigma_t^2 \mathbf{I}$ 为不可学习的常数（取 $\beta_t$ 或 $\tilde{\beta}_t$），只学习均值 $\mu_\theta$。

### 简化训练目标

原始变分下界（VLB）可以分解为各项 KL 散度之和。关键推导：最优均值 $\mu_\theta$ 的学习等价于预测噪声 $\boldsymbol{\epsilon}$：

$$\mu_\theta(\mathbf{x}_t, t) = \frac{1}{\sqrt{\alpha_t}}\left(\mathbf{x}_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}}\boldsymbol{\epsilon}_\theta(\mathbf{x}_t, t)\right)$$

由此导出**简化损失函数**：

$$L_{\text{simple}} = \mathbb{E}_{t, \mathbf{x}_0, \boldsymbol{\epsilon}}\left[\|\boldsymbol{\epsilon} - \boldsymbol{\epsilon}_\theta(\mathbf{x}_t, t)\|^2\right]$$

就是让网络预测加入的噪声 $\boldsymbol{\epsilon}$！训练极简：
1. 采样 $\mathbf{x}_0$
2. 随机选 $t \sim \text{Uniform}(1, T)$
3. 采样 $\boldsymbol{\epsilon} \sim \mathcal{N}(0, \mathbf{I})$
4. 计算 $\mathbf{x}_t = \sqrt{\bar{\alpha}_t}\mathbf{x}_0 + \sqrt{1 - \bar{\alpha}_t}\boldsymbol{\epsilon}$
5. 最小化 $\|\boldsymbol{\epsilon} - \boldsymbol{\epsilon}_\theta(\mathbf{x}_t, t)\|^2$

### 与得分匹配的联系

$\boldsymbol{\epsilon}_\theta(\mathbf{x}_t, t)$ 实质上是在预测得分函数（score function）的一个缩放版本：

$$\boldsymbol{\epsilon}_\theta(\mathbf{x}_t, t) = -\sqrt{1 - \bar{\alpha}_t} \nabla_{\mathbf{x}_t} \log q(\mathbf{x}_t)$$

因此训练扩散模型 = 训练去噪得分匹配 = 学习数据分布的梯度。这个联系使得 Langevin 动力学采样和扩散采样殊途同归。

### 网络架构

使用 U-Net（基于 PixelCNN++ 改造）：
- 下采样-上采样结构 + 跳跃连接
- 自注意力层在低分辨率特征图上
- 时间 $t$ 通过 Transformer 正弦位置编码 + MLP 注入每个残差块
- GroupNorm（而非 BatchNorm）
- 参数共享：同一个 $\boldsymbol{\epsilon}_\theta$ 处理所有 $t$

### 采样过程

从 $\mathbf{x}_T \sim \mathcal{N}(0, \mathbf{I})$ 开始，迭代 $T=1000$ 步：

$$\mathbf{x}_{t-1} = \frac{1}{\sqrt{\alpha_t}}\left(\mathbf{x}_t - \frac{\beta_t}{\sqrt{1 - \bar{\alpha}_t}}\boldsymbol{\epsilon}_\theta(\mathbf{x}_t, t)\right) + \sigma_t \mathbf{z}$$

其中 $\mathbf{z} \sim \mathcal{N}(0, \mathbf{I})$（$t=1$ 时 $\mathbf{z}=0$）。

## 三、实验结果

### 无条件图像生成

| 数据集 | FID | IS | 说明 |
|--------|-----|-----|------|
| CIFAR-10 | 3.17 | 9.46 | 当时不条件生成最佳 FID |
| LSUN Bedroom | 4.90 | - | 接近 ProgressiveGAN |
| LSUN Church | 7.89 | - | - |
| CelebA-HQ 256x256 | - | - | 高质量人脸生成 |

### 与 GAN 的对比

- DDPM 的 FID 不逊于 StyleGAN2，但**多样性显著更好**——GAN 模式崩塌问题在扩散模型中不存在
- 缺点：采样需要 1000 步前向传播，比 GAN 慢约 3 个数量级

### 消融实验关键发现

1. **简化损失 vs 完整 VLB**：简化损失训练更快、最终样本质量更好。VLB 的各时间步加权不均匀，简化损失相当于均匀加权，效果更好
2. **方差调度**：线性调度（$\beta_t$ 从 $10^{-4}$ 到 0.02 线性增加）效果稳定
3. **$\Sigma_\theta$ 是否学习**：固定方差 vs 学习方差在样本质量上差异不大，固定方差更简单
4. **反向过程方差选择**：$\sigma_t^2 = \beta_t$ 和 $\sigma_t^2 = \tilde{\beta}_t$ 结果几乎一样

### 渐进编码（Progressive Coding）

扩散模型天然是 lossy 压缩器：$\mathbf{x}_0$ 可以被编码为不同精度级别（对应不同 $t$），较早的 $t$ 编码低频信息，较晚的 $t$ 编码高频细节。

## 四、个人思考与影响

### 开创性影响

1. **统一了三条研究线**：扩散模型（Sohl-Dickstein et al., 2015）、得分匹配（Song & Ermon, 2019）、Langevin 动力学采样——本文证明它们本质上是同一件事
2. **简化目标是关键**：原始 VLB 训练效果很差，简化为预测噪声后立刻 work。这种"理论指导 -> 实践简化"的模式值得学习
3. **无需对抗训练**：避开了 GAN 的模式崩塌、训练不稳定等根本性困难，代价是采样慢

### 后续发展

- **采样加速**：DDIM（1000 步 -> 50 步）、DPM-Solver、一致性模型——采样速度从分钟级降到秒级
- **条件生成**：Classifier-free guidance 将扩散模型带入实用图像生成时代（DALL-E 2, Imagen, Stable Diffusion）
- **连续时间框架**：Score-based SDE 将离散扩散推广到连续，理论上更优雅
- **潜在扩散**：Latent Diffusion (Stable Diffusion) 在潜空间做扩散，大幅降低计算量

### 局限

- **采样速度**是原始 DDPM 最大的软肋（1000 步网络前向），后来被各种加速方法解决
- **线性噪声调度**不是最优的，后来 improved DDPM 提出余弦调度
- 论文没有做条件生成和引导采样，这些后来由 classifier guidance / classifier-free guidance 补上
- Log-likelihood 不如自回归模型和 flow 模型竞争

这篇论文证明了一个深刻道理：**好的数学框架 + 简化的实践目标 = 巨大影响力**。扩散模型的理论基础并不简单，但简化后的训练代码只有几十行，这种优雅让它迅速被社区接受。
