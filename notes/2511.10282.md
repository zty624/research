# Torch-Uncertainty: UQ Framework incl. Bayesian DL & VAE (Lafage et al., 2025)

**arxiv**: 2511.10282

---

## 一、理论推导（框架概述）

Torch-Uncertainty 是一个基于 PyTorch + Lightning 的不确定性量化 (UQ) 框架，不是单一理论论文，而是统一了 7 大类 UQ 方法的实现和评估：

1. **集成方法**：Deep Ensemble, Packed-Ensembles
2. **贝叶斯方法**：变分推断 (Bayes by Backprop), MC Dropout, SWAG, TRADI
3. **后校准**：Temperature Scaling, Laplace Approximation
4. **数据增强**：Test-Time Augmentation
5. **确定性方法**：Evidential Networks, Mean-Variance 输出头
6. **区间/保形预测**：Conformal Prediction
7. **高斯过程**：Deep Kernel Learning, SNGP

### 与 VAE 的关系

框架支持贝叶斯深度学习，包括变分推断（VAE 的核心方法）。VAE 的重参数化技巧、ELBO 优化等是贝叶斯方法的特例。框架中的 Bayesian Layers 可用于构建 VAE 的编码器/解码器。

### 评估指标

覆盖校准 (ECE)、OOD 检测 (AUROC/AUPR)、选择性分类、分布偏移鲁棒性等。

## 二、最小实验复现资源

| 项目 | 要求 |
|------|------|
| **框架** | PyTorch + Lightning, `pip install torch-uncertainty` |
| **任务** | 分类 (CIFAR-10/100, ImageNet), 回归, 语义分割 |
| **模型** | 预训练模型库 (HuggingFace), 即插即用 |
| **硬件** | 单 GPU 即可运行大部分实验 |
| **总估算** | **单 GPU, 安装即用** |

---

## 三、重要 Idea

1. **模块化 UQ 设计**：每种 UQ 技术作为独立模块，可自由组合。例如在 Bayesian 层上叠加 Packed-Ensembles，或在 Ensemble 上加 Conformal Prediction。这是区别于其他框架的核心优势。

2. **评估为中心**：不只是实现方法，更强调统一评估。多维度鲁棒性评估（校准、OOD、分布偏移、选择性分类），让方法比较更有意义。

3. **领域通用**：从视觉到时序，从分类到回归到分割，统一接口。大多数现有框架只覆盖部分任务。

4. **预训练模型库**：HuggingFace 上的 benchmarked 模型，降低 UQ 研究的入门门槛。

---

## 评价

作为软件框架论文，Torch-Uncertainty 的贡献是工程性的但很有价值：统一了碎片化的 UQ 实现，提供了公平比较的平台。对 VAE 研究者而言，框架中的 Bayesian Layers 和变分推断工具可直接用于 VAE 变体的实现和评估。不是理论突破，但大大降低了 UQ 研究的工程成本。
