**arxiv**: 1807.03039 | **关键词**: Normalizing Flow, 可逆生成模型, 1x1 可逆卷积, 精确似然 | OpenAI | NeurIPS 2018

## 一、核心问题

主流生成模型（VAE、GAN）存在各自的局限:
- **VAE**: 优化似然的下界 (ELBO)，而非精确似然
- **GAN**: 无法计算似然，训练不稳定，存在模式崩溃

基于 Normalizing Flow 的生成模型具有独特优势:
1. **精确对数似然**: 可直接优化和计算精确对数似然
2. **精确潜变量推断**: 可精确计算后验 $p(z|x)$
3. **训练与生成的并行化**: 双向计算均可高效进行

Glow 的核心贡献: 提出**可逆 1x1 卷积** (invertible 1x1 convolution) 作为 flow 的新构建模块，大幅简化了此前 RealNVP 的架构设计，并在标准基准上显著提升了对数似然性能。

## 二、方法

### 2.1 Normalizing Flow 基本框架

Flow-based 模型通过可逆变换 $f$ 将数据 $x$ 映射到潜变量 $z$:

$$z = f(x), \quad x = f^{-1}(z)$$

由变量替换公式，数据的对数似然为:

$$\log p(x) = \log p(z) + \log \left| \det \frac{\partial f}{\partial x} \right|$$

其中 $z = f(x)$，$\log p(z)$ 通常假设为标准正态分布。变换 $f$ 需要满足:
1. **可逆**: $f$ 和 $f^{-1}$ 都可高效计算
2. **雅可比行列式可高效计算**: 不需要 $O(D^3)$ 的行列式计算

### 2.2 Glow 的核心组件

Glow 由三种可逆变换的堆叠组成:

#### (1) Actnorm (Activation Normalization)

替代 Batch Normalization 的可逆归一化层:

$$y = s \odot x + b$$

其中 $s$ (scale) 和 $b$ (bias) 是可学习参数，初始化时使得第一个 mini-batch 的各通道均值为 0、方差为 1。与 BN 不同，actnorm 在推理时行为与训练时完全一致。

**对数行列式**: $\log |\det J| = h \cdot w \cdot \sum \log |s_i|$

#### (2) Invertible 1x1 Convolution (核心创新)

对形状为 $h \times w \times c$ 的张量，1x1 卷积等价于在每个空间位置应用相同的线性变换 $W \in \mathbb{R}^{c \times c}$:

$$y_{h,w} = W x_{h,w}$$

**可逆性**: 当 $W$ 可逆时，1x1 卷积可逆: $x_{h,w} = W^{-1} y_{h,w}$

**对数行列式**:

$$\log |\det J| = h \cdot w \cdot \log |\det W|$$

**参数化 $W$**: 使用 LU 分解保证可逆性和高效行列式计算:

$$W = P L (U + \text{diag}(s))$$

其中 $P$ 是固定排列矩阵，$L$ 是下三角矩阵（对角线为 1），$U$ 是对角线为 0 的上三角矩阵，$s$ 是向量。行列式为 $\log|\det W| = \text{sum}(\log|s|)$，计算复杂度从 $O(c^3)$ 降为 $O(c)$。

**与 RealNVP 的比较**: RealNVP 使用固定的通道翻转 (channel permutation) 来混合通道信息，而可逆 1x1 卷积是**可学习的**通道混合，更加灵活和高效。1x1 卷积本质上是置换操作的推广——当 $W$ 是置换矩阵时退化为固定排列。

#### (3) Affine Coupling Layer

继承自 RealNVP 的核心变换:

$$z_{1:d} = x_{1:d}, \quad z_{d+1:D} = x_{d+1:D} \odot s(x_{1:d}) + t(x_{1:d})$$

其中 $s(\cdot)$ 和 $t(\cdot)$ 是任意神经网络。NN() 的最后一层卷积**初始化为零**，使耦合层初始为恒等映射，有利于训练极深网络。

**对数行列式**: $\log |\det J| = \sum_{i=d+1}^{D} \log |s_i|$

### 2.3 整体架构

一个 flow step: Actnorm → Invertible 1x1 Conv → Affine Coupling Layer

使用 **多尺度架构**: 每隔若干步将一半通道直接输出作为最终潜变量，另一半继续通过后续 flow。共 $L$ 个 level，每 level 深度 $K$。

### 2.4 训练目标

最大化数据对数似然:

$$\mathcal{L} = \mathbb{E}_{x \sim p_{\text{data}}} \left[ \log p(z) + \log \left| \det \frac{\partial f}{\partial x} \right| \right]$$

## 三、实验结果

### 3.1 对数似然 (bits per dimension)

| 数据集 | RealNVP | Glow | 改进 |
|--------|---------|------|------|
| CIFAR-10 | 3.49 | **3.35** | -0.14 |
| ImageNet 32x32 | 4.28 | **4.09** | -0.19 |
| ImageNet 64x64 | 3.98 | **3.81** | -0.17 |
| LSUN (bedroom) | 2.72 | **2.38** | -0.34 |
| CelebA-HQ 256x256 | - | **1.03** | - |

### 3.2 消融: 1x1 卷积 vs. 固定排列

| 方法 | CIFAR-10 bpd |
|------|-------------|
| 固定排列 (RealNVP) | 3.49 |
| 随机排列 | 3.46 |
| 可逆 1x1 卷积 (Glow) | **3.35** |

可学习的 1x1 卷积比固定排列改善了 0.14 bpd，且 wallclock 时间仅增加约 7%。

### 3.3 潜变量操控

Glow 可以在潜空间中进行有意义的语义操控:
- **属性插值**: 在潜空间中线性插值，生成平滑过渡的图像
- **属性算术**: "微笑的人" - "不笑的人" + "不笑的人B" = "微笑的B"
- **温度调节**: 通过调节采样分布的温度，控制生成多样性和质量

### 3.4 大规模生成

在 CelebA-HQ 256x256 上训练的 Glow 模型能生成高分辨率、高质量的人脸图像，同时保持精确似然计算。

## 四、个人评价

**影响**: Glow 是 normalizing flow 发展史上的里程碑，证明了 flow-based 模型在精确似然优化下也能生成高质量图像。可逆 1x1 卷积成为后续 flow 模型的标准组件。

**优点**:
- 理论优雅: 精确似然、精确推断、双向高效计算
- 可逆 1x1 卷积简洁高效，替代了手工设计的通道排列
- 架构比 RealNVP 更简洁
- 生成的图像质量高，且支持潜空间操控

**局限性**:
- **计算代价高**: flow 需要逐步计算，训练和生成速度慢
- **表达能力**: 相比自回归模型，flow 的似然通常较差（PixelCNN 在 CIFAR-10 达到 2.92 bpd）
- **内存需求**: 可逆计算需要重建中间激活或使用 gradient checkpointing
- **对数似然与生成质量的矛盾**: 高似然不一定对应高质量生成

**后续发展**:
- **Flow++** (Ho et al., 2019): 可学习的逻辑混合 CDF 耦合层
- **Residual Flows** (Chen et al., 2019): Hutchinson 迹估计实现残差连接
- **FFJORD** (Grathwohl et al., 2019): 连续时间 normalizing flow (CNF)
- **Diffusion Models 的崛起**: DDPM/Score-based models 在生成质量上超越 flow，但 flow 在精确推断上仍有独特优势
