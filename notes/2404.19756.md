# KAN: Kolmogorov-Arnold Networks (Liu et al., 2024)

**arxiv**: 2404.19756 | **KAN 架构开山之作** | ICLR 2025

---

## 一、核心贡献

KAN（Kolmogorov-Arnold Networks）提出了一种替代 MLP 的新架构，灵感来自 Kolmogorov-Arnold 表示定理。核心贡献：

1. **可学习激活函数在边上（而非节点上）**：MLP 在节点上使用固定激活函数，KAN 在边上使用可学习的激活函数（参数化为 B 样条）
2. **准确性 + 可解释性**：在多个科学任务上，小规模 KAN 既比同规模 MLP 更准确，又能通过样条可视化直接"看到"学到的函数
3. **交互式模型精简**：通过样条剪枝、符号替换等步骤，可以从 KAN 中提取出符号公式
4. **KAT 近似定理**：证明了 KAN 在光滑 KA 表示存在时可战胜维度灾难，缩放指数 $\alpha = k+1 = 4$

## 二、关键技术点

### Kolmogorov-Arnold 表示定理

KAT（Kolmogorov-Arnold Theorem, 1957）指出：任何多元连续函数 $f: [0,1]^n \to \mathbb{R}$ 可以表示为：

$$f(\mathbf{x}) = \sum_{i=1}^{2n+1} \Phi_i\left(\sum_{j=1}^n \phi_{i,j}(x_j)\right)$$

其中 $\Phi_i: \mathbb{R} \to \mathbb{R}$ 和 $\phi_{i,j}: \mathbb{R} \to \mathbb{R}$ 都是单变量连续函数。

原始 KAT 是一个 2 层结构：内层 $n \times (2n+1)$ 个 $\phi_{i,j}$，外层 $2n+1$ 个 $\Phi_i$。

原始定理中的 $\phi_{i,j}$ 可能是不光滑甚至分形的，因此曾被认为是"理论上成立但实践上无用"的。本文的乐观态度：(1) 不必局限于 2 层 $(2n+1)$ 宽度的表示，可推广到任意宽度和深度；(2) 科学与日常中的函数通常具有稀疏组合结构和光滑性。

### KAN 架构

**从 KAT 到 KAN 的关键变化**：
1. 将 2 层扩展为任意深度（类比从 2 层感知机到深度 MLP）
2. 将宽度从 KAT 的 $2n+1$ 推广为任意宽度
3. 将单变量函数参数化为 B 样条

**KAN 层**：一个 KAN 层将 $n_{\text{in}}$ 维输入映射到 $n_{\text{out}}$ 维输出，层中共有 $n_{\text{in}} \times n_{\text{out}}$ 个可学习的单变量函数 $\phi_{i,j}$：

$$x_{l+1,j} = \sum_{i=1}^{n_{\text{in}}} \phi_{l,j,i}(x_{l,i})$$

一个 L 层 KAN：

$$\text{KAN}(\mathbf{x}) = (\Phi_{L-1} \circ \Phi_{L-2} \circ \cdots \circ \Phi_0)(\mathbf{x})$$

### 与 MLP 的对比

| 特性 | MLP | KAN |
|------|-----|-----|
| 激活函数位置 | 节点上（神经元） | 边上（连接） |
| 激活函数类型 | 固定（ReLU, GELU 等） | 可学习（B 样条） |
| 权重 | 线性变换矩阵 | 嵌入在样条参数中 |
| 参数效率 | 依赖宽度 | 依赖样条网格大小 |

MLP 将线性变换和非线性分离为 $W$ 和 $\sigma$，KAN 将它们统一在 $\Phi$ 中。

### B 样条参数化

每个可学习激活函数 $\phi(x)$ 表示为：

$$\phi(x) = w_b b(x) + w_s \text{spline}(x), \quad b(x) = \text{silu}(x) = \frac{x}{1+e^{-x}}$$

$$\text{spline}(x) = \sum_i c_i B_i(x)$$

- $b(x) = \text{silu}(x)$：基础函数，保证即使样条部分为零也有非零输出
- $B_i$ 是 B 样条基函数，$c_i$ 是可学习系数
- $w_b$（Xavier 初始化）、$w_s$（初始化为 1）：可学习的混合权重

**参数量**：深度 $L$、等宽 $N$、$G$ 间隔、$k$ 阶样条的 KAN 参数量为 $O(N^2 L(G+k))$。MLP 为 $O(N^2 L)$。KAN 参数更多，但通常需要更小的 $N$。

### KAT 近似定理与缩放律

**Theorem 2.1 (KAT)**：若 $f$ 有光滑 KA 表示，各 $\Phi_{l,i,j}$ 为 $(k+1)$ 阶连续可微，则：

$$\|f - \text{KAN}_G\|_{C^m} \leq C G^{-k-1+m}$$

当 $m=0$（$L_\infty$ 范数精度），RMSE 缩放指数为 $k+1$。使用 3 阶样条（$k=3$），$\alpha = k+1 = 4$。

**缩放律对比**：

| 理论 | 缩放指数 $\alpha$ | 依赖 |
|------|------------------|------|
| Sharma & Kaplan | $(k+1)/d$ | 维度 $d$（维度灾难） |
| Michaud et al. | $(k+1)/d^* = (k+1)/2$ | 最大 arity $d^* = 2$ |
| Poggio et al. | $m/2$ | 光滑度 $m$ |
| **KAN (本文)** | **$k+1 = 4$** | **独立于维度** |

KAT 与 UAT 的关键区别：KAT 利用函数的内在低维表示（KA 结构），UAT 不利用，因此受维度灾难影响。

### 网格扩展（Grid Extension）

MLP 需要从头训练更大模型来提升精度。KAN 可直接细化样条网格而无需从头重训：

1. 粗网格 $G_1$ 间隔上的 B 样条 $f_{\text{coarse}}(x) = \sum c_i B_i(x)$
2. 细网格 $G_2$ 间隔上拟合 $f_{\text{fine}}(x) = \sum c'_j B'_j(x)$，最小化：

$$\{c'_j\} = \arg\min_{\{c'_j\}} \mathbb{E}_x \left[ \left( \sum_j c'_j B'_j(x) - \sum_i c_i B_i(x) \right)^2 \right]$$

损失曲线呈现**阶梯状**：每次 grid extension 后损失急剧下降再平台期。最优测试损失在插值阈值附近（参数量约等于数据点数）。

**外部 vs 内部自由度**：计算图结构（节点连接方式）为外部自由度，样条网格点为内部自由度。KAN 同时拥有两者，外部负责学习组合结构，内部负责精确逼近单变量函数。

### 可解释性：稀疏化、剪枝、符号化

1. **稀疏化**：L1 范数 + 熵正则化

$$|\phi|_1 = \frac{1}{N_p} \sum_{s=1}^{N_p} |\phi(x^{(s)})|, \quad S(\Phi) = -\sum_{i,j} \frac{|\phi_{i,j}|_1}{|\Phi|_1} \log \frac{|\phi_{i,j}|_1}{|\Phi|_1}$$

$$\ell_{\text{total}} = \ell_{\text{pred}} + \lambda \left( \mu_1 \sum_l |\Phi_l|_1 + \mu_2 \sum_l S(\Phi_l) \right)$$

2. **剪枝**：节点级剪枝。节点 $(l, i)$ 的入/出分数定义为 $I_{l,i} = \max_k |\phi_{l-1,i,k}|_1$、$O_{l,i} = \max_j |\phi_{l+1,j,i}|_1$，若两者均低于阈值 $\theta = 10^{-2}$ 则剪除。

3. **符号化**：若激活函数疑似符号函数（如 $\sin$, $\log$），可设为符号形式 $y \approx c f(ax+b) + d$，通过迭代网格搜索 $a, b$ 和线性回归拟合 $c, d$。

4. **交互流程**：训练带稀疏化的 KAN -> 自动剪枝 -> 手动/建议符号化 -> 继续训练 -> 输出符号公式

### 持续学习

KAN 的样条具有局部性——改变某个样条区间只影响局部，不会像 MLP 全连接层那样产生全局干扰。在持续学习实验中，KAN 在学习新任务时不会遗忘旧任务，而 MLP 遭受严重灾难性遗忘。

## 三、实验结果

### 缩放律验证

KAN 在参数量-精度平面上几乎达到理论预测的 $\alpha = 4$ 缩放律（中位数 RMSE），而 MLP 即使在最慢的 $\alpha = (k+1)/d$ 线上也很快饱和。

关键观察：
- 更深的 KAN 比更宽的 KAN 表达力更强（如 [4,2,2,1] 远优于 [4,9,1]）
- 更小的 KAN 结构若恰好匹配函数结构，泛化更好（[2,1,1] 优于 [2,5,1]）

### 特殊函数拟合

15 个数学/物理特殊函数上，KAN 在 Pareto 前沿上一致优于 MLP。自动发现的紧凑 KAN 结构令人惊讶（如 Bessel 函数 $J_\nu(x)$ 只需 [2,2,1] 即可 RMSE < $10^{-2}$）。

### Feynman 数据集

Feynman 公式（来自《费曼物理学讲义》）上，KAN 在大部分公式上优于 MLP，且能自动发现与原公式一致的紧凑表示。

### 科学发现

**结理论（Knot Theory）**：
- 任务：从结的拓扑不变量预测签名（signature）
- [17,1,14] KAN 达到 81.6% 准确率，4 层宽度 300 的 MLP 仅 78%
- KAN 发现签名可通过 $\text{sign}(c_2) \cdot \text{sign}(\text{cusps})$ 近似，与数学家的猜想一致

**Anderson 局域化**：
- 任务：从势能参数预测是否发生局域化
- KAN 自动发现了区分局域化/非局域化的判据

### MNIST

KAN 和 MLP 准确率相近（~98%），KAN 略好。大规模视觉任务上 KAN 优势不明显。

## 四、个人思考与影响

### 核心洞察的价值

KAN 的核心洞察——**把可学习激活函数放在边上而不是节点上**——看似只是 MLP 的一个小改动，但蕴含深刻的含义：

1. **从全局近似到局部近似**：MLP 的权重矩阵做全局线性变换，激活函数做全局非线性；KAN 的每个连接都是独立的非线性映射，表达能力更灵活
2. **参数效率**：对光滑函数，KAN 的 B 样条比 ReLU 的分段线性更高效——少量样条节点就能精确表示
3. **可解释性不是附加的，而是内置的**：每个 $\phi_{i,j}$ 可以直接可视化，这比 MLP 的事后解释（如 SAE）自然得多

### 适用场景的局限性

KAN 的优势集中在中低维、光滑函数的拟合，这对科学发现（物理、数学公式发现）特别有吸引力。但：

1. **高维扩展性存疑**：KAT 本身是 $2n+1$ 宽度的 2 层结构，实际 KAN 做了推广但本质还是低维友好。图像、自然语言等高维数据不是 KAN 的主场
2. **训练速度**：B 样条的评估和反向传播比 ReLU 慢，大规模训练的效率是瓶颈
3. **与现有生态的兼容性**：Transformer + MLP 的训练管线极其成熟，KAN 需要重新设计优化器、初始化等
4. **"发现公式"需要人参与**：论文的符号回归流程是交互式的，不是全自动的

### 争议

KAN 发表后引发了不小的争议：
- **KAT 的重要性被高估？**：实际 KAN 并未使用原始 KAT 的 $2n+1$ 结构，更像是"受 KAT 启发的广义网络"。有批评指出这和更早的可学习激活函数工作区别不大
- **实验规模太小**：所有"优越性"都在小规模任务上验证，没有在大规模基准上展示
- **与 MLP 的对比不够公平**：MLP 使用 LBFGS 训练且未使用现代技术（如 Adam + LR schedule、dropout、batch norm），可能低估了 MLP 的表现
- **缺乏大规模实用性验证**：能否在 Transformer 的 FFN 层中替换 MLP 并取得提升？这个问题在论文中未回答，后来社区的工作也未能证明其在大规模上的优势

### 我的判断

KAN 的真正价值不在于"替代 MLP"（在大多数场景下不会），而在于**为科学发现提供了一种新工具**：
- 物理学家和数学家可以用 KAN 从数据中发现符号关系
- 样条可视化让人可以"看到"模型学到了什么
- 交互式精简让黑箱变白箱

对于通用深度学习（大规模视觉、语言、生成模型），KAN 目前不是 MLP 的替代品。但对于"从数据中发现数学结构"这一特定任务，KAN 提供了一个有价值的思路。它的长期影响可能类似于 PINN——不是通用架构，但在特定领域有不可替代的价值。
